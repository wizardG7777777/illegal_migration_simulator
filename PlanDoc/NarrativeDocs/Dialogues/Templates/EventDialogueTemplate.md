# 事件对话标准格式模板

> 本文档提供标准化的对话文案格式模板，供文案撰写人员参考。

---

## 1. 标准事件结构模板

### 1.1 固定事件模板

```markdown
### 【事件名称】(ID: `act{X}_{category}_{name}`)

**你看到了什么**

[50-100字场景描述，第二人称，冷静克制的旁观者视角]

用黑色幽默或现实主义的笔触描绘场景，营造氛围，提供决策背景。避免直接说明数值变化，专注于感官体验和情绪渲染。

---

**可选行动**

**选项 A：选项描述（10-20字）**
- 条件：无 / 体魄 ≥ X / 需道具：[道具标签]
- 消耗：行动点 X，金钱 XX
- 结果：[30-50字结果描述，体验化表达数值变化]
- 数值效果：
  - 现金 +XX / -XX
  - 身体健康度 +XX / -XX
  - 心理健康度 +XX / -XX
  - 属性成长：[属性] +X（概率/一定）

**选项 B：选项描述（10-20字）**
- 条件：无 / 智力 ≥ X / 需道具：[道具标签]
- 消耗：行动点 X
- 结果：[30-50字结果描述]
- 数值效果：[同上]

**选项 C（解锁/隐藏）：选项描述（10-20字）**
- 解锁条件：[槽位匹配到特定道具/属性达标]
- 结果：[30-50字结果描述，包含额外收益]

---

**槽位文案映射**（内部参考）

| 槽位类型 | 匹配道具 | 文案表现 |
|---------|---------|---------|
| [slot_id] | [道具名称] | [对应的描述文本] |
```

```markdown
---

**事件结果**

[描述发生的事情和强制性的后果]

- 固定效果：
  - 身体健康度 +/-XX
  - 心理健康度 +/-XX
  - 现金 +/-XX
  - 获得/失去道具：[道具名称]

**玩家应对**（如有选择）

**选项 A：应对方式**
- 条件：[属性 ≥ X / 持有道具]

**选项 B：应对方式**
- 条件：无
- 结果：[接受后果的描述]
```

## 2. 文案风格示例

### 示例 1：打工事件（送外卖）

```markdown
### 送外卖 (ID: `act1_work_delivery`)

**强制槽位**：必须持有 `transport` 标签道具（电动车/自行车/摩托车）

**你看到了什么**

手机屏幕在支架上闪烁，新订单的提示音响起。你戴上头盔，拧动电门，汇入夜晚的车流。这个城市从不睡觉，而你也是其中之一。

街边的霓虹灯在头盔面罩上投下斑驳的光影。你计算着时间——这一单如果能在30分钟内送达，可以多拿2块钱奖励。

---

**可选行动**

**选项 A：正常送单**
- 条件：无
- 消耗：行动点 2，身体健康度 8，心理健康度 2
- 结果：你稳稳当当地跑了6单，没有超时也没有投诉。收入比快递分拣高不少，但腿酸得不行。
- 数值效果：
  - 现金 +120 CNY
  - 身体健康度 -8
  - 心理健康度 -2

**选项 B：抢时间送单**
- 条件：体魄 ≥ 8
- 消耗：行动点 2，身体健康度 12
- 结果：你闯红灯、抄近路，终于在倒计时结束前送达。顾客给了好评，但你的心脏跳得厉害。
- 数值效果：
  - 现金 +160 CNY
  - 身体健康度 -12
  - 体魄 +1（概率）

**选项 C：多平台同时跑（需道具：[高性能电动车]）**
- 解锁条件：持有 `transport` 标签的高优先级道具
- 结果：你同时挂着三个平台的账号，手机一直在响。收入翻倍，但你几乎没有喘息的时间。
- 数值效果：
  - 现金 +200 CNY
  - 身体健康度 -15
  - 心理健康度 -5

---

**槽位文案映射**

| 槽位类型 | 匹配道具 | 文案表现 |
|---------|---------|---------|
| transport_slot | 电动车 | "电动车电量充足，轻松应对晚高峰。" |
| transport_slot | 自行车 | "自行车慢是慢点，但不用充电，随时可以出发。" |
| transport_slot | 无 | "【灰色不可点击】你需要一辆代步工具才能接单。" |
```

### 示例 2：随机事件（暴雨）

```markdown
### 暴雨袭击 (ID: `rand2_storm`)

**触发时机**：回合开始时自动触发
**触发权重**：20
**冷却回合**：3 回合
**场景限制**：仅场景2

---

**你遇到了什么**

天空毫无征兆地裂开一道口子，暴雨倾泻而下。你们前一秒还在赶路，下一秒就成了落汤鸡。向导骂着脏话，说这是雨季的第一场，你们的运气"真他妈好"。

---

**事件结果**

暴雨持续了一整夜。你们挤在一棵树下，雨水顺着领口灌进去，浑身湿透。没有遮蔽的夜晚让你的身体备受煎熬。

- 固定效果：
  - 身体健康度 -15
  - 行动点 -1（本回合）

**免疫条件**：持有 [sleeping_bag] 睡袋
- 免疫描述："幸好你带了睡袋，虽然还是湿透，但至少保住了一点体温。"

---

**玩家应对**

**选项 A：咬牙坚持**
- 条件：体魄 ≥ 8
- 结果：你的身体底子不错，虽然难熬，但还是挺过来了。
- 数值效果：
  - 身体健康度 -10（替代 -15）
  - 体魄 +1（小概率，极限生存的经验）

**选项 B：寻找遮蔽**
- 条件：风险意识 ≥ 6
- 结果：你注意到不远处有一个岩洞，带着大家转移过去。虽然还是冷，但比淋雨强多了。
- 数值效果：
  - 身体健康度 -8（替代 -15）
  - 风险意识 +1（积累经验）

**选项 C：接受现实**
- 条件：无
- 结果：你缩在树下，祈祷这场雨快点过去。湿冷慢慢侵蚀着你的身体。
- 数值效果：
  - 身体健康度 -15（标准值）
```

## 4. 场景语气差异说明

### 4.1 场景1：国内准备

**基调**：现实主义、疏离感、黑色幽默
**情绪色彩**：焦虑与期待并存、对未来的不确定
**常用句式**：
- "你站在[地点]，看着[景象]"
- "这一天和之前没什么两样"
- "你开始怀疑[想法]"
- "夜深了，你还在[动作]"

**文案示例**：
```
你站在便利店的冰柜前，看着里面整齐排列的饮料。
一瓶可乐五块钱，够你在食堂吃顿饱的。
你关上柜门，转身继续搬货。
```

### 4.2 场景2：跨境穿越

**基调**：紧张、生存压力、原始恐惧
**情绪色彩**：恐惧、绝望、偶尔的希望
**常用句式**：
- "前方是[险境]，身后是[绝境]"
- "夜幕降临时，[危险]"
- "向导说[警告]，你心里[反应]"
- "你们沉默地[动作]，只有[声音]"

**文案示例**：
```
夜幕降临，你们终于找到了一处洼地休息。
向导点燃一支烟，火光在黑暗中格外刺眼。
"明天要过河，"他说，"水里有鳄鱼，看你们的运气。"
你抱紧背包，试图入睡。
```

### 4.3 场景3：美国生存

**基调**：荒诞、讽刺、慢性焦虑
**情绪色彩**：被追捕的紧张、生存的疲惫、黑色幽默
**常用句式**：
- "你看着[账单/新闻]，心里[计算/恐惧]"
- "又一天过去了，你[动作]"
- "据说[传言]，你不知道真假"
- "在这个[地点]，你学会了[技能]"

**文案示例**：
```
手机屏幕亮了，是房东催租的短信。
你数了数钱包里的钞票，刚好够这个月的房租。
但这也意味着接下来两周你只能吃泡面。
你叹了口气，打开打工软件，看看今晚有没有临时单。
```

## 5. 特殊事件类型文案指南

### 5.1 终结态进入事件

```markdown
### 进入【濒死】状态 (ID: `term{1/2/3}_dying_entry`)

**发生了什么**

你的身体已经到了极限。每一次呼吸都变得艰难，视线边缘开始发黑。
你意识到，如果不立刻采取措施，可能撑不了多久了。

**倒计时提示**（根据体魄决定 0-3 回合）

- 体魄 0-1："你已经无法动弹，意识正在逐渐模糊。"
- 体魄 2-8："你还有 [X] 回合的时间，必须立刻找到救治方法。"
- 体魄 9-15："你还有 [X] 回合的时间，情况危急但尚有希望。"
- 体魄 16-20："你的身体素质让你比常人多争取了一些时间，还有 [X] 回合。"

**可选行动**
- 使用医疗道具（如有）
- 寻找帮助（特定事件）
- 听天由命（游戏结束）
```

## 6. 文案检查清单

撰写完成后，请检查以下项目：

- [ ] 事件ID与 `DesignDocs/Events/` 中的定义一致
- [ ] 场景描述字数在 50-100 字范围内
- [ ] 每个选项文本长度在 10-20 字
- [ ] 每个结果描述在 30-50 字
- [ ] 数值变化已使用体验化描述包装（不直接显示"+10"）
- [ ] 使用第二人称"你"
- [ ] 语气符合当前场景基调
- [ ] 没有直接说教或道德评判
- [ ] 错别字和标点符号检查通过
