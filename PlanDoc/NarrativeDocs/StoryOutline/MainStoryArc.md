# 主线剧情大纲

> 本文档定义《去美国》文字冒险游戏的整体叙事架构，包含故事概述、三幕结构、核心冲突与主题隐喻。

---

## 故事概述

你，一个普通的中国年轻人，厌倦了日复一日的生活。"美国梦"的幻象在社交媒体上闪耀——自由、财富、机会。你决定铤而走险，踏上一条没有退路的旅程。从国内准备到跨境穿越，再到美国境内的艰难求生，每一步都是对意志与运气的考验。最终，你会拿到那张象征"胜利"的绿卡，还是倒在追逐梦想的路上？

**核心体验**："梗游戏式"的短回合高情绪强度，用冷静克制的文字讲述一个关于欲望、代价与归属的黑色幽默故事。

## 三幕结构

### 第一幕：与过去告别（场景1 - 国内准备）

| 属性 | 内容 |
|-----|------|
| **主题** | 【与过去告别】离开熟悉的一切，为未知的旅程做准备 |
| **预期回合** | 25-50回合（软限制30回合触发"放弃"结局） |
| **核心玩法** | 打工攒钱、能力提升、寻找离境途径 |
| **环境压力** | 【就业冰河期】——经济下行，打工收入下降，就业困难 |

#### 故事弧

```
开场          上升           催化剂          转折点
 │             │              │               │
 ▼             ▼              ▼               ▼
┌───┐      ┌────────┐     ┌────────┐     ┌────────┐
│日常│  →  │打工攒钱│  →  │触发决定│  →  │告别出发│
│平淡│      │能力提升│     │离境决心│     │踏上旅途│
└───┘      └────────┘     └────────┘     └────────┘
```

**开场**：你坐在出租屋的床上，手机屏幕亮着——又是一个留学移民成功案例。你决定不再只是看看而已。

**上升**：你开始打工、健身、学英语、收集信息。每一天都在为那个"决定"积累筹码。

**催化剂**：某个触发事件（可能是一笔意外之财，可能是一次中介的"可靠"承诺，也可能是对未来的恐惧终于压倒了犹豫），你下定决心——走。

**转折点**：最后一次与家人通话，最后一次在国内过年，最后一次看这片熟悉的土地。你背起行囊，走向未知。

#### 情绪曲线

```

  │   ╱      ╲    决心
  │  ╱        ╲  ╱
  │ ╱          ╲╱
```

**情绪走向**：从焦虑（对现状的不满和对未来的恐惧）逐渐攀升至决心（做出了决定后的释然与坚定）。

### 第二幕：生存与抉择（场景2 - 跨境穿越）

| 属性 | 内容 |
|-----|------|
| **主题** | 【生存与抉择】极端环境下的求生与人性考验 |
| **预期回合** | 5-15回合 |
| **核心玩法** | 路线选择、资源管理、向导依赖 |
| **高危地区前进带来的身心压力** | 【法律真空】——旅途艰辛，在高危地区前进且缺乏法律保障，身体和心理健康会持续下降 |

#### 故事弧

```
进入新世界     探索适应       中点转折        危机升级       转折点
    │            │            │              │             │
    ▼            ▼            ▼              ▼             ▼
┌────────┐  ┌────────┐  ┌────────────┐  ┌────────┐  ┌────────┐
│进入边境│ →│选择路线│ →│同伴变故/   │ →│环境极限/│ → │越过边境│
│地区    │  │遭遇环境│  │向导背叛    │  │致命抉择│    │到达美国│
└────────┘  └────────┘  └────────────┘  └────────┘  └────────┘
```

**进入新世界**：你来到了边境地区，一切都和国内不同。语言不通、环境恶劣、处处是危险。

**探索适应**：你必须做出第一个重大选择——跟随向导胡安？雇佣走私客阿米尔？还是独自穿越？

**中点转折**：旅程过半时，发生了一件改变一切的事。也许是同伴的背叛，也许是向导的卷款潜逃，也许是你发现了一个"捷径"。

**危机升级**：身体达到极限，物资耗尽，边境巡逻队在身后紧追不舍。每一次选择都可能是最后一次。

**转折点**：终于，边境线就在眼前。向导带你找到漏洞、你徒手攀上边境墙、或者奇迹般地穿越沙漠——你进入了美国。

### 第三幕：身份与归属（场景3 - 美国生存）

| 属性 | 内容 |
|-----|------|
| **主题** | 【身份与归属】在结构性压迫中寻找出路 |
| **预期回合** | 60+回合 |
| **核心玩法** | 现金流管理、身份获取、多维度生存 |
| **环境压力** | 【大通胀时代】+ 【移民执法压力】——现金消耗递增、突击检查概率上涨 |
| **通关条件** | 获得绿卡（合法身份） |

#### 故事弧

```
进入目标地      建立基础        身份追寻        高潮          结局
    │            │             │             │            │
    ▼            ▼             ▼             ▼            ▼
┌────────┐  ┌──────────┐  ┌──────────┐  ┌────────┐  ┌────────┐
│初到美国│ →│找住处/   │ →│庇护/婚姻/│ →│关键抉择│ →│命运落定│
│身无分文│  │打工/社交 │  │人才/投资 │  │成功/失败│  │结局触发│
└────────┘  └──────────┘  └──────────┘  └────────┘  └────────┘
```

**进入目标地**：你终于来到了梦想之地，却发现现实与想象相去甚远。没有身份、没有积蓄、没有人脉——你只是一个非法入境者。

**建立基础**：从最底层开始。睡家庭旅馆、送外卖、打黑工。你需要建立最基本的生存网络。

**身份追寻**：你开始探索获得合法身份的途径：
- **庇护之路**：编造或真实的政治迫害故事，面对移民法官
- **婚姻移民**：是真爱还是交易？无论如何，你需要一个美国公民
- **杰出人才**：证明你对美国有"非凡价值"
- **投资移民**：需要你没有的50万美元
- **抽签移民**：数千万分之一的概率，命运之神会眷顾你吗？

**高潮**：某个关键抉择点——也许是庇护面试的那一天，也许是移民局突击检查你的住所，也许是你终于攒够了申请费。成败在此一举。

**结局**：你的旅程迎来终点。绿卡、黑户、遣返、入狱、或者死亡——命运已经写好。

## 核心冲突

### 内在冲突：追求梦想 vs 安全舒适

| 冲突面 | 表现 |
|-------|------|
| **追求梦想** | 对美国的向往、对现状的不满、对"更好生活"的执念 |
| **安全舒适** | 国内的安稳生活、家人的牵挂、对未知的恐惧 |

**叙事呈现**：
- 场景1的"放弃"结局线——承认"留在国内也是一种选择"
- 场景2的"想要放弃"事件——崩溃状态下的自我怀疑
- 场景3的"思乡/孤独发作"——对过去的怀念与对现状的质疑

### 外在冲突：非法身份 vs 生存需求

| 冲突面 | 表现 |
|-------|------|
| **非法身份** | 无法合法工作、随时可能被遣返、没有基本权利保障 |
| **生存需求** | 需要收入维持生活、需要住所遮风挡雨、需要医疗保障健康 |

**叙事呈现**：
- 场景3的现金流恶性循环——越穷越要打工，越打工身体越差，身体越差越需要钱
- 假证件的风险——没有证件寸步难行，但假身份随时可能被识破
- 突击检查的恐惧——【移民执法压力】Debuff 带来的持续心理压迫

### 人际冲突：信任 vs 背叛

| 冲突面 | 表现 |
|-------|------|
| **信任** | 向导的承诺、同伴的互助、老乡的支援 |
| **背叛** | 向导卷款潜逃、同伴出卖、老乡诈骗 |

**叙事呈现**：
- 场景2的"同伴背叛"随机事件——社交属性不足时的风险
- 场景2的"遭遇武装抢劫"——同为偷渡者的加害
- 场景3的"移民局突袭"——信任网络也可能暴露

## 主题与隐喻

### 主题一：美国梦

**核心问题**："美国梦"是真实的机遇，还是精心包装的幻象？

**叙事呈现**：
- 场景1：社交媒体上的成功案例 vs 现实中打工攒钱的艰辛
- 场景2：对"到达美国"的幻想 vs 穿越过程中的残酷现实
- 场景3：梦想之地的光鲜 vs 黑户生活的窘迫

**玩家反思**：通过角色经历的反差，让玩家自己思考——追逐的到底是什么？值得吗？

### 主题二：身份认同

**核心问题**：没有合法身份，你还是"人"吗？

**叙事呈现**：
- 场景3的道具驱动生活成本系统——没有证件就无法租房、无法开户、无法正常生活
- 假证件事件——用谎言换取"正常人"的生活
- 彻底黑户结局——成为没有身份、没有记录、没有过去的人

**玩家反思**：身份是什么？一张卡片？社会认可？还是自我认同？

### 主题三：结构性压迫

**核心问题**：个人的失败，有多少是个人选择，有多少是结构的不公？

**叙事呈现**：
- 场景3的【大通胀时代】Debuff——无论你怎么努力，生活成本都在上涨
- 特朗普政策压力事件——政治环境的变化直接影响生存
- 现金流恶性循环——努力工作却越陷越深

**玩家反思**：当系统本身在挤压你时，"努力就能成功"的叙事还成立吗？

## 叙事节奏说明

### 回合节奏设计

| 场景 | 回合数 | 节奏特点 | 叙事密度 |
|-----|-------|---------|---------|
| 场景1 | 25-50 | 相对舒缓，允许反复打工 | 中密度，日常感 |
| 场景2 | 5-15 | 紧凑高压，几乎每回合都有危机 | 高密度，紧迫感 |
| 场景3 | 60+ | 长期消耗，慢性压力积累 | 中密度，疲惫感 |

### 事件触发节奏

- **场景1**：固定事件为主（打工、读书），随机事件为辅（降租、被骗）
- **场景2**：随机事件密度最高（暴雨、野兽、背叛），强化不可控感
- **场景3**：事件链为主（庇护申请、婚姻流程），随机事件为辅（突击检查、账号被封）

### 情绪节奏控制

- **场景1**：焦虑→决心（单线上扬）
- **场景2**：希望→绝望→坚韧（V型反转）
- **场景3**：迷茫→起伏→确定（波浪式前进）
