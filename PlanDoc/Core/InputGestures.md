# 手势操作与快捷键规范

本文档定义《去美国》游戏的交互输入规范，包括移动端手势操作和桌面端键盘快捷键。

**设计原则**：
- 移动端：手势直观、符合直觉、避免误触
- 桌面端：快捷键高效、符合习惯、便于单手操作

---

```
[静止] → 手指按下 → [检测中]
              ↓
        水平移动>50px ──→ [左滑/右滑状态]
              ↓                      ↓
        垂直移动>30px      松开后执行对应操作
              ↓                      ↓

              ↓
        放弃手势识别
```

**防误触机制**：
- 垂直移动优先：当垂直位移>30px时，取消水平手势识别
- 速度阈值：滑动速度需>100px/s才触发快捷操作
- 回弹动画：松开手指后0.3s弹性回弹

**撤销栈实现**：
```typescript
// 伪代码
interface UndoAction {
  type: 'EXECUTE_EVENT' | 'USE_ITEM' | 'ADJUST_SLOT';
  prevState: GameState;
  description: string; // 用于显示"撤销了：送外卖"
}

const undoStack: UndoAction[] = [];
const MAX_UNDO = 5;

function pushUndo(action: UndoAction) {
  if (undoStack.length >= MAX_UNDO) {
    undoStack.shift();
  }
  undoStack.push(action);
}

function undo() {
  if (undoStack.length === 0) return;
  const action = undoStack.pop();
  restoreState(action.prevState);
  showToast(`已撤销：${action.description}`);
}
```

## 3. 输入冲突处理

| 冲突场景 | 处理方案 |
|---------|---------|
| 快捷键与输入框冲突 | 输入框聚焦时禁用快捷键（除Esc外） |
| 手势与滚动冲突 | 垂直滚动优先，水平触发手势阈值 |
| 长按与拖拽冲突 | 500ms内无移动视为长按，否则视为拖拽 |
| 双指缩放与双指滑动冲突 | 判断两指距离变化率，>10%视为缩放 |
| 双击与单击冲突 | 等待300ms确认无第二次点击后执行单击 |
| 多手势同时触发 | 按优先级处理：点击 > 长按 > 滑动 > 缩放 |

### 3.1 输入框聚焦状态

```typescript
// 伪代码
let isInputFocused = false;

// 输入框聚焦时
inputElement.onfocus = () => {
  isInputFocused = true;
  // 禁用所有快捷键，但保留Esc
};

// 输入框失焦时
inputElement.onblur = () => {
  isInputFocused = false;
  // 恢复快捷键
};

// 键盘事件处理
document.onkeydown = (e) => {
  if (isInputFocused && e.key !== 'Escape') {
    return; // 不处理
  }
  // 正常处理快捷键...
};
```

### 3.2 手势优先级队列

```
手势识别优先级（从高到低）：

1. 单击/双击
   └── 等待300ms判断是否为双击
   
2. 长按
   └── 500ms后触发，期间移动<10px
   
3. 垂直滚动
   └── 垂直位移>30px，用于叙事文本
   
4. 水平滑动
   └── 水平位移>50px，用于事件卡片
   
5. 双指缩放
   └── 双指距离变化>10%

   └── 顶部区域+垂直下拉>100px

```


**焦点管理实现**：
```typescript
// 伪代码
const focusableSelectors = [
  'button:not([disabled])',
  'a[href]',
  'input:not([disabled])',
  'select:not([disabled])',
  'textarea:not([disabled])',
  '[tabindex]:not([tabindex="-1"])'
];

function getFocusableElements() {
  return document.querySelectorAll(focusableSelectors.join(','));
}

function handleTabKey(e) {
  const focusables = getFocusableElements();
  const currentIndex = Array.from(focusables).indexOf(document.activeElement);
  
  if (e.shiftKey) {
    // Shift+Tab - 反向
    const prevIndex = currentIndex > 0 ? currentIndex - 1 : focusables.length - 1;
    focusables[prevIndex].focus();
  } else {
    // Tab - 正向
    const nextIndex = currentIndex < focusables.length - 1 ? currentIndex + 1 : 0;
    focusables[nextIndex].focus();
  }
  e.preventDefault();
}
```
