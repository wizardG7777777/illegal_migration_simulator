# 场景2详细地点设定

> 本文档定义场景2（跨境穿越）的完整地理空间和地点系统。
> 基于 SceneSystemArchitecture.md 中的 Act2State 路线设计。

---

## 1. 边境地区概览

### 1.1 地理范围

场景2覆盖从中南美洲出发地到美国边境的完整穿越路线，包括：

- **出发区域**：墨西哥南部/中美洲国家边境
- **过渡区域**：墨西哥境内陆路通道
- **危险走廊**：墨西哥北部贩毒集团控制区
- **边境区域**：美墨边境（最终目标）

### 1.2 路线概览

```
出发小镇 → 雨林区域 → 河流/沼泽 → 沙漠区域 → 边境墙
    ↓           ↓           ↓           ↓          ↓
  补给准备    湿热危险    水路抉择    干旱考验   最终跨越
```

## 2. 主要地点节点

### 2.1 出发小镇（Departure Town）

**基本信息**
| 属性 | 值 |
|-----|---|
| **ID** | `act2_loc_departure` |
| **类型** | 安全区（无死亡事件） |
| **阶段** | preparation |

**描述**：
> 塔帕丘拉，恰帕斯州边缘的混乱小镇。蛇头在街角抽烟，汽车旅馆里挤满了来自世界各地的人——海地人、古巴人、中国人。空气中弥漫着柴油和绝望的味道。

**可触发事件**：
| 事件类型 | 事件ID | 说明 |
|---------|-------|-----|
| 固定 | `act2_buy_supplies` | 购买补给 |
| 固定 | `act2_hire_guide` | 雇佣向导 |
| 固定 | `act2_rest_inn` | 旅馆休息 |
| 固定 | `act2_gather_intel` | 收集路线情报 |
| 随机 | `rand2_meet_companion` | 遇到潜在同伴 |

**特色地点**：
- **汽车旅馆**：$10/晚，健康+10，心理+5
- **杂货店**：基础补给，价格×1.2
- **蛇头聚集点**：可购买向导服务（$200-500）
- **移民救助站**：免费食物和水，但有ICE线人风险

**离开条件**：
- 持有至少3天水/食物
- 或雇佣向导（可豁免）

---

### 2.2 雨林区域（Rainforest Zone）

**基本信息**
| 属性 | 值 |
|-----|---|
| **ID** | `act2_loc_rainforest` |
| **类型** | 危险区 |
| **阶段** | journey |

**描述**：
> 树冠遮天蔽日，湿度让衣服永远不干。每一步都可能惊动毒蛇，每一口呼吸都带着霉菌和腐烂植物的味道。在这里，GPS是块废铁，唯一的方向感来自向导的记忆。

**环境Debuff**：
```yaml
name: 雨林湿热
healthChange: -3/回合
mentalChange: -2/回合
effects:
  - 装备损耗加速
  - 【疟疾】风险（未服用抗疟疾药时）
```

**可触发事件**：
| 事件类型 | 事件ID | 危险等级 |
|---------|-------|---------|
| 随机 | `rand2_mosquito_swarm` | ★★ |
| 随机 | `rand2_snake_encounter` | ★★★ |
| 随机 | `rand2_malaria_risk` | ★★★ |
| 随机 | `rand2_flash_flood` | ★★★★ |
| 固定 | `act2_find_water` | 安全 |
| 固定 | `act2_harvest_food` | 安全 |

**致命危险**：
- **疟疾**：未服药时20%几率感染
- **毒蛇**：风险意识<8时触发遭遇
- **洪水**：暴雨后触发，可冲走物资

**生存策略**：
- 必备：抗疟疾药、防水装备
- 建议：当地向导（免疫迷路事件）

---

### 2.3 河流/沼泽（River & Swamp）

**基本信息**
| 属性 | 值 |
|-----|---|
| **ID** | `act2_loc_river` |
| **类型** | 抉择区（路线分支） |
| **阶段** | journey |

**描述**：
> 乌苏马辛塔河在雨季变成狂暴的巨龙。走私客的充气筏要价$50，而游过去只需要胆量——或者愚蠢。

**路线选择**：
| 路线 | 消耗 | 风险 | 奖励 |
|-----|------|-----|------|
| 桥梁 | $20过桥费 | ★ | 无 |
| 绕路 | +3回合 | ★★ | 发现隐藏补给点 |
| 走私筏 | $50 | ★★ | 节省时间 |
| 游泳 | 0 | ★★★★★ | 无 |

**致命危险**：
- **鳄鱼**：游泳/涉水时触发（见 Act2_DeathEvents.md）
- **急流**：游泳时90%死亡率
- **蚂蟥**：沼泽区域健康-2/回合

**可触发事件**：
| 事件类型 | 事件ID | 说明 |
|---------|-------|-----|
| 固定 | `act2_cross_bridge` | 付费过桥 |
| 固定 | `act2_cross_raft` | 走私筏 |
| 固定 | `act2_cross_swim` | 游泳（高危） |
| 随机 | `rand2_river_current` | 被冲走风险 |
| 随机 | `rand2_find_boat` | 发现废弃小船 |

### 2.4 沙漠区域（Desert Zone）

**基本信息**
| 属性 | 值 |
|-----|---|
| **ID** | `act2_loc_desert` |
| **类型** | 极限危险区 |
| **阶段** | border_approach |

**描述**：
> 索诺拉沙漠。白天50度能把人烤干，夜里10度能把人冻僵。仙人掌是唯一的生命，而它们也在嘲笑你的到来。这是最后也是最残酷的考验。

**环境Debuff**：
```yaml
name: 极端干旱
healthChange: -5/回合（无水时-15）
mentalChange: -4/回合
effects:
  - 水消耗量×3
  - 夜间体温骤降
  - GPS干扰（沙暴时）
```

**可触发事件**：
| 事件类型 | 事件ID | 危险等级 |
|---------|-------|---------|
| 随机 | `rand2_dehydration` | ★★★★★ |
| 随机 | `rand2_heat_stroke` | ★★★★ |
| 随机 | `rand2_sandstorm` | ★★★ |
| 随机 | `rand2_coyote_attack` | ★★ |
| 固定 | `act2_day_travel` | 风险 |
| 固定 | `act2_night_travel` | 较低风险但迷失概率+ |

**致命危险**：
- **脱水**：无水时3回合内死亡（见 Act2_DeathEvents.md）
- **中暑**：白天强行军时触发
- **沙暴**：丢失方向，物资损耗

**生存策略**：
- 必备：水袋（3L以上）、盐片、睡袋
- 建议：夜间行进，白天隐蔽
- 向导：可减少50%迷路概率

**特殊地点**：
- **废弃水塔**：20%几率有水（随机）
- **仙人掌林**：可获取水分（生存技能≥8）
- **边境巡逻废弃哨所**：可能有遗留物资

### 2.5 边境墙（The Wall）

**基本信息**
| 属性 | 值 |
|-----|---|
| **ID** | `act2_loc_border_wall` |
| **类型** | 终局区（场景转换点） |
| **阶段** | crossing |

**描述**：
> 它就在那里——9米高，带刺铁丝网，探照灯每30秒扫过一遍。墙的另一边，就是美国。自由，或者死亡。

**穿越选项**：
| 方式 | 要求 | 费用 | 成功率 | 失败后果 |
|-----|------|-----|-------|---------|
| 向导带路 | 持有guide道具 | $500 | 80% | 受伤，可重试 |
| 寻找漏洞 | 发现漏洞事件 | $0 | 50% | **死亡**（被射杀） |
| 攀爬翻越 | 体魄≥12 | $0 | 40% | **死亡**（坠落） |
| 奇迹事件 | 随机触发 | $0 | 5% | **死亡**（被捕杀） |

**可触发事件**：
| 事件类型 | 事件ID | 说明 |
|---------|-------|-----|
| 固定 | `act2_cross_guide` | 向导带路 |
| 固定 | `act2_cross_gap` | 找漏洞（高危） |
| 固定 | `act2_cross_climb` | 攀爬（高体魄要求） |
| 随机 | `rand2_border_patrol` | 遭遇巡逻队 |
| 随机 | `rand2_border_miracle` | 奇迹事件 |

**成功结果**：
- 进入场景3
- 根据穿越方式给予不同起始包

**失败结果**：
- 向导路径：受伤，返回沙漠
- 其他路径：**死亡结局**

## 3. 路线设计图

### 3.1 北方路线（Northern Route）

```
出发小镇 → 雨林(3-5天) → 高原(2-3天) → 沙漠(2-4天) → 边境墙
   ↓           ↓              ↓            ↓            ↓
 安全区    疾病/迷路      海拔适应     极端干旱     最终考验
```

**特点**：
- 总耗时：7-12天
- 死亡率：中等（35%）
- 补给点：3个
- 推荐：有向导时选择

### 3.2 南方路线（Southern Route）

```
出发小镇 → 沿海公路(2-3天) → 沼泽(3-4天) → 河流(1-2天) → 边境墙
   ↓           ↓               ↓            ↓            ↓
 安全区    检查站风险      湿地疾病     涉水危险     最终考验
```

**特点**：
- 总耗时：6-10天
- 死亡率：中高（45%）
- 补给点：4个
- 推荐：英语好、社交高时选择

### 3.3 沿海路线（Coastal Route）

```
出发小镇 → 海岸线(4-6天) → 渔港(1天) → 海路(??) → 美国海岸
   ↓           ↓            ↓          ↓          ↓
 安全区    暴露/沙暴      补给/休息   极端危险    偷渡船
```

**特点**：
- 总耗时：不确定
- 死亡率：极高（60%）
- 补给点：2个
- 推荐：高风险偏好，有海路向导时

## 4. 特殊地点

### 4.1 补给营地（Supply Camp）

| 属性 | 值 |
|-----|---|
| **ID** | `act2_loc_supply_camp` |
| **类型** | 休息点 |

**描述**：
> 前人的足迹留下的营地。篝火余烬还未完全熄灭，可能有人不久前还在这里。

**功能**：
- 免费休息（健康+10，心理+5）
- 可能发现遗留物资（随机）
- 可能遇到其他走线者（社交事件）

### 4.2 废弃建筑（Abandoned Structure）

| 属性 | 值 |
|-----|---|
| **ID** | `act2_loc_abandoned` |
| **类型** | 探索点 |

**描述**：
> 边境附近的废弃农舍，窗户破碎，门廊坍塌。蛇头说这里曾经是安全屋，直到ICE开始钓鱼执法。

**功能**：
- 避雨/避暑（环境Debuff暂停）
- 搜寻物资（可能发现食物、水、药品）
- 风险：被占领、陷阱、或者其他走线者的敌意

### 4.3 检查站（Checkpoint）

| 属性 | 值 |
|-----|---|
| **ID** | `act2_loc_checkpoint` |
| **类型** | 危险点 |

**描述**：
> 墨西哥军警的检查站。他们不在乎你去美国，但他们在乎你有没有带钱。

**事件**：
- 贿赂通过（$20-100）
- 逃跑（体魄检定）
- 被拘留（损失时间，可能被遣返）

## 6. 设计与叙事要点

1. **渐进压迫**：从出发小镇的安全感，到边境墙的绝望压力

2. **路线选择**：没有"正确"路线，只有"适合当前属性"的路线

3. **死亡即终点**：场景2没有存档点，死亡后从场景1重新开始

4. **向导价值**：向导不仅是道具，更是叙事元素和难度调节器

5. **环境叙事**：每个地点的描述都在讲述"非法移民"的集体记忆
