# 场景3详细地点设定

> 本文档定义场景3（美国生存）的主要城市和地点系统。
> 基于 SceneSystemArchitecture.md 中的 Act3State 生活成本系统设计。

---

## 1. 概述

场景3的地点设计遵循以下原则：
- **系统性生存**：地点与现金流、身份获取、社会网络深度绑定
- **风险分层**：不同地点有不同的被查/被抢风险
- **华人社区**：作为主要社会支持网络
- **真实映射**：基于真实美国城市的华人社区和非法移民生态

**城市概览**

## 6. 城市选择策略

### 6.1 洛杉矶优势
- 庇护城市，ICE执法较松
- 华人社区成熟，不说英语也能活
- 气候好，无极端天气成本
-  Uber机会多

### 6.2 纽约优势
- 庇护申请律师资源集中
- 现金工机会多（装修、美甲）
- 公共交通便利（无车也能活）
- 文化资源丰富（心理恢复）

### 6.3 选择建议

| 玩家类型 | 推荐城市 | 原因 |
|---------|---------|-----|
| 英语差 | 洛杉矶 | 华人社区更大 |
| 想走庇护 | 纽约 | 律师资源多 |
| 有车/会开车 | 洛杉矶 | Uber机会多 |
| 高体魄 | 纽约 | 外卖/装修工收入高 |
| 高社交 | 任意 | 人脉网络都有效 |
| 高智力 | 纽约 | 庇护申请成功率+ |

## 7. 地点与系统关联

### 7.1 现金流系统

```
居住地月租 + 食品成本 + 交通成本 = 基础生存成本
打工收入 - 基础生存成本 = 可用于身份申请的储蓄
```
