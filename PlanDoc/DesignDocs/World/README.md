## 使用说明

- **文案写作参考**：撰写事件描述时参考此处设定，保持世界观一致性
- **梗与金句来源**：现实中的电子宠物事迹、论坛黑话等素材
- **背景考据**：美国移民政策、跨境路线、地下经济等现实依据

## 与 NarrativeDocs/ 的区别

- **World/**: 支撑材料、考据、设定参考（工具性质）
- **NarrativeDocs/**: 实际游戏内文本（故事线、对话、结局等）
