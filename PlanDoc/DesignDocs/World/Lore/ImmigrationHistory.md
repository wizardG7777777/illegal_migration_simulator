# 美国移民政策背景（Lore）

## 概述

本文档提供美国移民政策的历史背景和现实考据，为文案写作提供参考。

---

## 历史脉络

### 1980年代：移民改革与控制法案 (IRCA)
- 允许约270万非法移民获得合法身份
- 首次要求雇主验证员工身份
- 开启了"大赦"先例

### 1996年：非法移民改革与移民责任法案 (IIRIRA)
- 大幅收紧了移民政策
- 扩大了驱逐出境的范围
- 增加了对非法入境的惩罚

### 2001年后：国土安全部成立
- 9/11后移民管控全面收紧
- ICE (移民与海关执法局) 成立
- 边境安全成为国家安全议题

### 特朗普时期 (2017-2021)
- "零容忍"政策
- 旅行禁令
- 公共负担规则 (Public Charge Rule)
- 对庇护申请的严格限制

### 现状
- 约1100万非法移民生活在美国
- 加州、德州、纽约、佛州为主要聚集地
- 华人非法移民主要途径：签证逾期、偷渡、庇护申请

## 关键术语

| 术语 | 说明 | 游戏内用途 |
|-----|------|-----------|
| ICE | 移民与海关执法局 | 场景3突击检查的主要执行者 |
| CBP | 海关与边境保护局 | 场景2边境巡逻 |
| EAD | 工卡 (就业授权文件) | 场景3重要道具 |
| SSN | 社会安全号码 | 打工必备 |
| A Number | 外国人登记号码 | 庇护申请相关 |
| 递解令 | Removal Order | 被抓住后的最坏结果 |
| 工卡 | Work Permit | 合法打工的凭证 |

## 华人移民地下经济

### 主要行业
- 中餐馆（后厨、送外卖）
- 装修业
- 美甲店
-  Uber/Lyft
- 仓库/快递分拣

### 身份 workaround
- 借用他人 SSN
- 现金支付
-  subcontractor (独立承包商) 模式
-  "ghost" payroll
