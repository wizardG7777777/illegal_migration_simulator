# 角色信息
## 主要消耗资源
**消耗资源会随着游戏的推进消耗或者增加，波动可能会比较剧烈，也是玩家在游玩过程中的主要压力来源**
- 身体健康度：角色的健康资源，执行大部分高强度事件时需要消耗这个值。归零时持续进行身体健康判定，判定失败则判定角色死亡。
- 心理健康度：角色的精神资源，大部分事件对这个值没有要求，不会消耗，但是一旦这个值归零，就会持续性的触发高负面收益的事件。高负面收益包括虚拟角色死亡。
- 金钱资源：很多事件都需要消耗金钱，也是在全场景下都必须要进行管理的资源种类：
 - 美元：初始为0，第一幕完全不会消耗，第二幕和第三幕的事件作为主要的货币进行消耗。
 - 人民币：创建角色时会被赋予的角色资源，在第一幕事件中作为主要的货币进行消耗，可以在第一幕中的一些事件中兑换为美元。
- 行动点：一回合能够执行的行动的抽象代表，其每回合开始阶段都会尝试恢复到最大上限值。但是藉由随机事件和消耗性道具可以恢复到超过最大值。当然，debuff 也可以使得这个恢复不能恢复到最大值。
### 终结状态机（身体健康度 / 心理健康度归零）

#### 终结态总则（共同规则）
- **触发条件**：任一资源降至 `<= 0`，立刻进入对应终结态：
  - 身体健康度 `<= 0` → 进入【濒死】
  - 心理健康度 `<= 0` → 进入【崩溃】
  - 资金`<= 0` → 进入【匮乏】
#### 【匮乏】状态 （资金 <= 0）
- 场景1下，人民币<=0 时进入此判定
- 场景2和场景3下，美元<=0 时进入此判定
- 负面效果很简单，每回合扣除身体健康度和心理健康度。不会造成死亡。
#### 【濒死】终结态（身体健康度 <= 0）
- **压迫规则**：进入终结态后，本回合与后续回合将被“终结流程”接管：
  - 若进入终结状态，则每回合开始时不会触发随机事件，只会触发终结事件。根据角色的生存属性确定倒计时回合数。
    - **公式**：终结回合 = clamp( floor((P + 6) / 7), 0, 3 )，终结回合只是一个整数，最小值为0，没有显式的最大值设定。
  - **行动点被严重限制**（无法回到上限），并且每回合开始额外扣除 1 点行动点（代表呼吸、躲避、硬撑的成本），等差数列累加，直到最后角色什么都做不了，只能等死。
- **倒计时与结局**：终结状态算是给玩家的一个警告，警告玩家好好管理角色的资源。同时也是对那些增加生存属性的抉择的奖励，因为生存属性的增长可以延长这个喘息窗口的长度。

- **进入濒死（即时效果）**
  - 立刻触发“濒死冲击”强制事件：行动点归零（或最低为 0），并额外扣除少量金钱/物资（代表急救材料、止血布、求水求药）。
  - 记录状态：`终结态=濒死`，`倒计时=1回合`，如果倒计时回合数大于计算公式得出的上限，则角色被判定为死亡。
- **濒死的滚雪球惩罚（压迫感来源）**
  - 若本回合结束时身体仍 `<= 0`：
    - 进入下一回合时给角色添加 debuff（你在失血、发烧、抽筋），并额外扣除心理健康度（因为恐惧与痛觉）。
- **脱离濒死（唯一出口）**
  - 当身体健康度被恢复到 `> 0` 时，终结态解除，濒死状态下获得的 debuff 不会自动消失，提醒玩家“下次不会这么幸运”。

#### 【崩溃】终结态（心理健康度 <= 0）
- **压迫规则**：进入终结态后，本回合与后续回合将被“终结流程”接管：
  - 若进入终结状态，则每回合开始时不会触发随机事件，只会触发终结事件。根据角色的智力属性确定倒计时回合数（智力越高，窗口越短）。
    - **公式**：崩溃窗口回合 = clamp( 3 - floor(I / 10), 1, 3 )
      - I=0 → 3 回合；I=20 → 1 回合；最小值固定为 1
  - **行动点被严重限制**：每回合开始时行动点恢复上限被压低（你恢复不了正常状态），并额外扣除 1 点行动点（过度思考、惊恐反刍、注意力涣散的成本），等差数列累加，直到你几乎无法做出任何有效行动。
- **倒计时与结局**：崩溃不是“惩罚”，是“判决的宣读”。你必须在倒计时耗尽之前让心理健康度恢复到 `> 0`，否则游戏直接结束。

- **进入崩溃（即时效果）**
  - 立刻触发“崩溃冲击”强制事件：在以下后果中随机选择一个
    - 随机丢失一个道具
    - 百分比降低随机一项可消耗资源
    - 降低一个随机的属性值5点
  - 记录状态：`终结态=崩溃`，`倒计时=崩溃窗口回合`。从0开始计数，直到倒计时大于窗口回合数为止。
- **崩溃回合流程（每回合开始强制执行）**
  每回合开始的时候触发一次“崩溃冲击”强制事件。

- **崩溃的滚雪球惩罚（压迫感来源）**
  - 若本回合结束时心理仍 `<= 0`：
    - 倒计时 `-1`，并在下一回合开始前强制追加一次“高负面收益事件”（通常是有自毁倾向的事件）。
  - 若倒计时归零后心理仍 `<= 0`：**直接游戏结束**（自毁性死亡 / 被捕 / 失踪，以结局文案呈现）。

- **脱离崩溃（唯一出口）**
  - 当心理健康度被恢复到 `> 0` 时，终结态解除，但崩溃状态下丢失的属性或者资源不会恢复，提醒玩家“你只是暂时活过来了”。
## 主要属性值
- 体魄：代表角色体能
- 智力：代表角色的学习能力
- 英语技能：代表角色的英语沟通能力
- 社交：代表角色探听到市井消息的能力，社交能力越高，角色越有机会探听到一些有利于自己的随机事件
- 风险意识：代表角色能够意识到风险，或者提前预知后果的能力
- 生存能力：代表角色在极端情况下能够存活的能力，同时也代表了角色钻制度空子的能力

### 体魄
- 获取的方式：体力劳动、训练类行动、长期坚持的里程碑事件、少数增益型常驻道具
- 获取的量级：常规+1；里程碑+2；极少数重大事件+3；严重受伤/过劳-1~-3
- 主要的适用范围：高强度行动、极端环境消耗、长距离移动、负重与疾病相关事件
- 该属性是如何影响事件的触发与进展的：降低高强度事件的身体消耗，提升体力检定成功率，解锁“硬扛/带人同行”等分支，并决定濒死终结态的喘息窗口长度

### 智力
- 获取的方式：学习类行动（路线研究/规则学习/技能学习）、付费课程、关键任务里程碑
- 获取的量级：常规+1；付费或里程碑+2；长期睡眠不足/精神压力导致的负面事件-1（或短期debuff）
- 主要的适用范围：信息处理、骗局识别、流程理解、材料准备与策略规划类事件
- 该属性是如何影响事件的触发与进展的：提高识别骗局与止损选项的触发概率与成功率，并用于崩溃终结态的窗口计算（智力越高窗口越短：崩溃窗口回合=clamp(3-floor(I/10),1,3)）

### 英语技能
- 获取的方式：语言学习行动、沉浸式环境成长（特定工作/社区）、面试或关键对话的里程碑
- 获取的量级：稳定成长以+1为主；里程碑+2；少数挫败/压力事件可造成-1或短期debuff
- 主要的适用范围：求职/租房/就医/律师沟通、盘查询问、正规渠道信息获取与办理流程
- 该属性是如何影响事件的触发与进展的：作为关键选项门槛与检定加成，提升“正规渠道机会事件”的权重，并将部分失败代价从“扣大量金钱/热度暴涨”替换为“扣行动点/小幅代价”

### 社交
- 获取的方式：主动社交行动（混圈/加入群聊/打工结识）、完成他人委托与互助、关系型常驻道具与事件
- 获取的量级：常规+1；关系里程碑+2；口碑崩坏/背叛/被出卖-1~-3
- 主要的适用范围：信息与人脉获取、落脚点与工作推荐、危机求助、灰色渠道的引荐与交易
- 该属性是如何影响事件的触发与进展的：提高“额外随机事件”与“信息机会事件”的出现概率，同时在高热度下也更容易引来麻烦，并解锁推荐、借宿、内部消息等分支

### 风险意识
- 获取的方式：路线研究与复盘、谨慎决策累积、经历险情但幸存的成长、部分道具增益
- 获取的量级：常规+1；重大险情后+2；长期依赖酒精/逃避导致-1或短期debuff
- 主要的适用范围：高风险选项的后果判断、避坑、降低随机性带来的毁灭性结果
- 该属性是如何影响事件的触发与进展的：用于“后果揭示”检定（提前看见代价类型或风险提示），降低暴死型事件的发生频率，并将部分失败降级为“付出代价逃离”的分支

### 生存能力
- 获取的方式：第二幕极端环境中的正确决策、掌握灰色技巧、关键道具与关键角色带来的长期增益
- 获取的量级：常规+1；重大节点+2；极少数传奇机会+3；被抓/重伤/丢关键道具-1或触发长期debuff
- 主要的适用范围：极端环境存活、躲避搜捕、资源匮乏下的应对、制度缝隙与灰区操作
- 该属性是如何影响事件的触发与进展的：提升极端事件的成功率并降低失败代价，解锁“用技巧换资源/绕行”分支，并在终结态中提供少数“把立刻死亡改为严重后遗症”的保命路径
