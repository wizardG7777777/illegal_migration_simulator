# 经济系统（Economy System）

## 概述

《去美国》的经济系统模拟非法移民在三个阶段的资金流动：
- **场景1（国内）**：人民币积累，为出境做准备
- **场景2（跨境）**：美元消耗，生存挑战
- **场景3（美国）**：美元收支平衡，长期生存压力

---

## 核心设计哲学

### "现金流断裂"体验
场景3的核心压力来源于：**每月固定支出不可逃避，收入却充满不确定性**。

这种设计模拟现实焦虑：
- 房租/食物/交通是刚性的
- 工作收入受身份限制、突击检查、账号被封等影响
- 玩家必须在"省钱"和"冒险"之间做选择

### 货币转换
| 场景 | 货币 | 说明 |
|-----|------|------|
| 场景1 | 人民币（CNY） | 无法直接使用，需兑换 |
| 场景2 | 美元（USD） | 场景1可兑换，场景主要货币 |
| 场景3 | 美元（USD） | 唯一货币 |

**汇率设计**：固定汇率 1 USD = 7 CNY（简化，不考虑波动）

## 场景1：国内准备期

### 收入来源

| 事件ID | 工作类型 | 收入范围 | 属性要求 | 风险 |
|-------|---------|---------|---------|------|
| `act1_work_delivery` | 快递分拣 | ¥80-120/次 | 体魄≥3 | 低 |
| `act1_work_supermarket` | 超市夜班 | ¥150-200/次 | 体魄≥5 | 低 |
| `act1_work_construction` | 工地短包 | ¥300-500/次 | 体魄≥8 | 中（受伤风险）|

### 支出类型

| 类型 | 支出项 | 金额 | 说明 |
|-----|-------|------|------|
| 固定 | 房租 | ¥1500/月 | 为了玩法简化，不考虑租金上涨 |
| 固定 | 食物 | ¥600/月 | 场景1食物成本固定 |
| 可选 | 书籍 | ¥150/本 | 全局书籍池 |
| 可选 | 健身卡 | ¥800/月 | 解锁健身房训练 |
| 可选 | 美元兑换 | 按需 | 汇率固定 7 |

### 经济循环设计

```
初始资金: ¥3000-5000（根据角色背景）
    ↓
打工攒钱 ──▶ 兑换美元 ──▶ 购买书籍/道具 ──▶ 提升属性
    ↑                                              │
    └──────────────── 更好工作机会 ◀───────────────┘
```

### 目标数值
- **通关场景1建议资金**：$2000-5000（¥14,400-36,000）
- **理想准备期**：15-25 回合
- **资金不足风险**：场景2生存难度急剧上升

## 场景2：跨境穿越期

### 资金特性

场景2是**纯消耗期**，几乎没有收入来源。

### 主要支出

| 类型 | 支出项 | 金额范围 | 说明 |
|-----|-------|---------|------|
| 必需 | 向导费用 | $500-2000 | 可选（自己穿越省但风险高）|
| 必需 | 食物/水 | $5-20/天 | 持续消耗 |
| 必需 | 住宿/露营 | $0-30/天 | 野外露营免费但有debuff |
| 可选 | 装备补给 | $50-200 | 购买/升级生存装备 |
| 随机 | 贿赂/过路费 | $100-500 | 遭遇检查时 |

### 生存成本估算

| 路线类型 | 预计天数 | 最低成本 | 舒适成本 |
|---------|---------|---------|---------|
| 独自穿越（雨林） | 7-14天 | $100-200 | $300-500 |
| 向导带路 | 5-10天 | $800-2500 | $1500-3500 |
| 走私客路线 | 3-7天 | $1500-4000 | $2000-5000 |

### 资金耗尽后果

进入【匮乏】状态，触发场景2专属匮乏事件：
- `term2_desperation`：出卖身体/尊严换取食物
- `term2_steal`：偷窃其他偷渡者（高风险被抓）
- `term2_abandoned`：被同伴抛弃（倒计时-1）

## 场景3：美国生存期

### 生活成本系统

场景3引入**月度固定支出**机制，是核心压力来源。

#### 成本结构

```
月度总支出 = 食品成本 + 住宿成本 + 出行成本
```

| 成本项 | 基础金额 | 道具影响 | 通胀影响 |
|-------|---------|---------|---------|
| 食品 | $400/月 | 无（人活着就要吃饭）| 受 economic debuff 影响 |
| 住宿 | $0-2500/月 | lodging 标签道具决定 | 受 economic debuff 影响 |
| 出行 | $50-600/月 | transport 标签道具决定 | 受 economic debuff 影响 |

#### 住宿道具

| 道具ID | 类型 | 月成本 | 心理/晚 | 备注 |
|-------|------|-------|--------|------|
| （无） | 露宿街头 | $0 | -5 | 获得【露宿街头】debuff |
| `key_shelter` | 收容所床位 | $0 | -2 | 不稳定，可能被赶走 |
| `key_blackhotel` | 家庭旅馆 | $600 | +3 | 场景3起步选择 |
| `key_apartment` | 合租房间 | $800 | +4 | 正规租房 |
| `lease_apartment` | 公寓 | $1200 | +5 | 需要 fake_lease |
| `key_house` | House合租 | $2500 | +6 | 最高舒适度 |

#### 出行道具

| 道具ID | 类型 | 月成本 | 收入影响 | 备注 |
|-------|------|-------|---------|------|
| （无）| 步行 | $0 | 只能找附近工作 | 严重限制工作选择 |
| `card_bus` | 公交卡 | $100 | 正常 | 基础选择 |
| `vehicle_ebike` | 电动车 | $50 | 收入×0.9 | 低成本选择 |
| `vehicle_used` | 二手车 | $350 | 正常 | 性价比选择 |
| `vehicle_tesla` | 特斯拉 | $600 | 收入×1.3 | Uber升级选项 |

### 通胀系统

#### 触发方式
通过 `RANDOM` 类型事件触发 `economic` 类型的环境 Debuff。

#### Debuff 结构

```typescript
interface EconomicDebuff {
  type: 'economic';
  subtype: 'usd_inflation';
  effects: {
    foodMultiplier: number;      // 食品通胀倍率
    lodgingMultiplier: number;   // 住宿通胀倍率  
    transportMultiplier: number; // 出行通胀倍率
  };
  duration: number;  // -1 表示永久，直到被其他事件解除
}
```

#### 示例事件：油价危机

```yaml
事件: rand3_oil_crisis
触发:
  type: RANDOM
  weight: 15
  scenes: [act3]
效果:
  添加Debuff:
    name: "出行成本飙升"
    type: economic
    effects:
      transportMultiplier: 1.4  # 出行成本+40%
    duration: -1  # 永久，需特定事件解除
文案: "油价暴涨，加满一箱油多花$50"
```

#### 通胀率等级

| 等级 | 倍率范围 | 描述 |
|-----|---------|------|
| 正常 | 1.0 | 无通胀 |
| 轻度 | 1.0-1.3 | 生活成本上升30% |
| 中度 | 1.3-1.7 | 生活压力明显 |
| 严重 | 1.7-2.5 | 生存危机边缘 |
| 恶性 | >2.5 | 几乎无法生存 |

### 收入来源

#### 打工事件收入表

| 事件ID | 工作类型 | 基础收入 | 证件要求 | 风险 | 备注 |
|-------|---------|---------|---------|------|------|
| `act3_work_delivery` | 送外卖 | $40-60/次 | 可选 | 低 | 英语提升 |
| `act3_work_kitchen` | 餐厅后厨 | $80-100/次 | 无 | 低 | 体力消耗 |
| `act3_work_uber` | Uber司机 | $120-150/次 | 必需 | 中 | 有封号风险 |
| `act3_work_supermarket` | 超市打工 | $100-120/次 | 必需 | 低 | 稳定但收入一般 |

#### 收入修正因素

| 因素 | 修正 | 说明 |
|-----|------|------|
| 交通工具 | ×0.9-1.3 | 根据 vehicle_slot 道具 |
| 身份凭证 | +50% | 有 identity_slot 道具 |
| 英语技能 | +10%/级 | 英语≥8时解锁高小费选项 |
| 执法压力 | -10%~-30% | pressure Debuff 影响 |
| 通胀 Debuff | ×倍率 | 经济型 Debuff 影响 |

### 月度收支模拟

#### 理想情况（有证件+车）

```
收入:
  Uber司机 × 8次 = $120 × 8 = $960
  修正: ×1.3(特斯拉) = $1248

支出:
  食品: $400
  住宿: $1200 (公寓)
  出行: $600 (特斯拉)
  ─────────────────
  总计: $2200

结余: -$952 (需要更多工作次数或降低标准)
```

#### 起步情况（无证+家庭旅馆）

```
收入:
  送外卖 × 10次 = $50 × 10 = $500
  修正: ×0.9(电动车) = $450

支出:
  食品: $400
  住宿: $600 (家庭旅馆)
  出行: $50 (电动车)
  ─────────────────
  总计: $1050

结余: -$600 (资金持续流失，必须寻找更高收入工作)
```

## 经济压力曲线设计

### 场景3资金曲线目标

```
资金
  │
  │    ╭─────── 理想玩家（收支平衡或盈余）
  │   ╱
  │  ╱  ╭────── 普通玩家（缓慢流失，有压力但可控）
  │ ╱  ╱
  │╱  ╱   ╭─── 困难玩家（快速流失，濒临崩溃）
  ├───────────────
  └──────────────────▶ 回合数
```

### 关键设计参数

| 参数 | 目标值 | 说明 |
|-----|-------|------|
| 月度收支平衡工作次数 | 8-12次 | 玩家需要规划如何分配行动点 |
| 起步资金消耗速度 | 10-15回合耗尽 | 给玩家时间建立收入来源 |
| 通胀事件频率 | 每20-30回合 | 制造中期压力波峰 |
| 执法压力对收入影响 | 10%-30% | 显著的负面影响但不致命 |

## 数值平衡检查清单

### 场景1
- [ ] 普通玩家20回合可攒够 $3000
- [ ] 高体魄玩家可通过工地快速攒钱但有健康风险
- [ ] 高智力玩家可通过识别骗局减少损失
- [ ] 资金不足的玩家场景2有明显劣势（增加难度选择感）

### 场景2
- [ ] 最低成本路线仍需要 $500 生存
- [ ] 向导路线成本 $1500-3000 合理
- [ ] 资金耗尽后有匮乏事件但不立即致死

### 场景3
- [ ] 起步玩家（$1000+家庭旅馆+电动车）可生存10-15回合
- [ ] 理想玩家（证件+好车+公寓）可实现收支平衡
- [ ] 通胀事件+执法压力叠加时制造严重危机
- [ ] 庇护申请费用 $1000-3000 是中期目标
