# 场景切换机制（Scene Transition）

## 概述

场景切换是《去美国》三幕结构的关键节点。玩家通过特定事件触发场景切换，每次切换都意味着游戏节奏、核心玩法和压力的显著变化。

**场景流向**（单向）：
```
场景1（国内准备）──┬──▶ 场景2（跨境穿越）──▶ 场景3（美国生存）
                    │
                    └──▶ 场景3（美国生存）【直飞】
```

---

## 设计意图

### 1. 门槛与风险的平衡

场景切换没有"奖励"，只有门槛与风险的权衡：

| 切换路径 | 门槛 | 风险 | 结果 |
|---------|------|------|------|
| 场景1 → 2（走线） | 极低（只需钱） | **极高**（穿越雨林+边境） | 非法入境 |
| 场景1 → 3（留学） | 高（学费+材料） | 低 | 合法入境 |
| 场景1 → 3（旅游） | 极高（面签通过） | 低 | 合法入境 |
| 场景2 → 3（向导） | 中（付费） | 中 | 非法入境 |
| 场景2 → 3（漏洞） | 低（需发现） | **极高（致死）** | 非法入境 |
| 场景2 → 3（攀爬） | 低（体魄≥12） | **极高（致死）** | 非法入境 |
| 场景2 → 3（随机） | 极低 | **极高（致死）** | 非法入境 |

**核心设计**：走线是最便宜的方式，但必须经历场景2的高死亡率；直飞签证昂贵且门槛高，但跳过场景2直接开始场景3。

### 2. 零和重新开始
场景切换时**清空常驻道具栏**，制造"断舍离"的压迫感：
- 场景1辛苦攒钱买的电动车带不走
- 场景2穿越用的指南针到场景3无用
- 唯一保留：全局书籍池（知识是最轻的资产）

### 3. 叙事锚点
每个切换事件都是关键剧情节点，应该：
- 有强烈的仪式感
- 呈现选择的不可逆
- 用文案强化"无法回头"的情绪

## 场景1 → 场景2：走线

### 触发事件

#### 事件ID: `act1_transition_walk`

**事件描述：**
> 你在网上联系到了一个叫"李哥"的人。他说只要交一笔钱，就能安排你从厄瓜多尔出发，经巴拿马雨林、墨西哥，最后到达美国边境。这条路被称为"走线"，是近年来最流行的非法入境方式。

**触发条件：**
- 无硬性要求（最低门槛路径）
- 可选：持有 `contact` 标签道具（如蛇头名片）可降低费用

**可选行动：**

| 选项 | 条件 | 费用 | 结果 |
|-----|------|------|------|
| **答应下来** | 现金 ≥ ¥15,000 | ¥15,000-25,000 | 进入场景2，获得基础生存包 |
| **讨价还价** | 社交 ≥ 6 | ¥12,000-18,000 | 社交检定，成功则降价，失败则原价 |
| **再等等** | - | - | 关闭事件，可再次触发 |

**场景切换结算：**
```yaml
扣除: 人民币（根据选项）
清空: 所有常驻道具栏（场景1道具）
给予:
  - 美元现金: $500-1000（取决于剩余人民币兑换）
  - 基础生存包:
      - 水袋 × 1
      - 干粮 × 3
      - 基础指南针 × 1
保留:
  - 全局书籍池（已购买的书籍）
  - 角色属性值
场景状态初始化:
  - 回合数: 继承累计
  - 场景特定状态: 场景2初始状态
```

### 切换后文案

```
你最后一次回望这座城市的灯火。
明天，你将踏上一段没有退路的旅程。
李哥说，快则两周，慢则一月。
他说这话的时候，眼睛里没有光。
```

## 场景1 → 场景3：签证直飞

### 路径A：旅游签证

#### 事件链: `act1_visa_tourist`

**触发条件：**
- 持有 `document` 标签道具（如假房产证、假在职证明）
- 或 智力 ≥ 8（能编织可信的出行理由）

**事件I：准备面签**
> 美国大使馆门口排着长队。你手里攥着准备好的材料，心跳得厉害。面签官以严厉著称，据说他一分钟就能决定一个人的命运。

**选项：**
1. **用假材料蒙混过关**（需 `fake_documents`）
2. **靠口才说服**（智力 ≥ 8）
3. **找中介培训**（现金 -¥3000，获得面签技巧 buff）

**事件II：面签当天**（2-3回合后触发）

面签官提问（随机3选1）：
- "你在中国的收入是多少？"
- "你在美国有亲戚吗？"
- "你计划在美国停留多久？"

**检定机制：**
```
成功条件（满足其一）：
- 持有高质量假材料 + 智力 ≥ 6
- 智力 ≥ 10
- 有风险意识 ≥ 7 且选择了保守回答

失败结果：
- 拒签
- 获得 debuff 【签证黑名单】（无法再次申请旅游签）
```

**成功后的场景切换：**
```yaml
给予:
  - 旅游签证（道具，有效期6个月）
  - 机票（场景3起始事件触发）
  
清空: 所有常驻道具栏
保留:
  - 全局书籍池
  - 角色属性值
  
场景3起始状态:
  - 合法入境身份（旅游签）
  - 现金: $2000-5000（根据剩余人民币）
  - 初始道具: 基础住宿（旅馆预订）
  - 特殊: 6回合倒计时（签证逾期倒计时）
```

### 路径B：野鸡大学（灰色签证工厂）

#### 事件链: `act1_visa_student`

**背景说明：**
这是一所纯灰色的"签证工厂"，不提供任何实际教学，唯一的存在价值是发放I-20表格以获取学生签证。你需要定期向学校"注册"并交纳学费来维持这个虚假的学生身份。

**触发条件：**
- 持有 `fake_diploma` 或 `networking_letter`
- 或 完成 `act1_apply_college` 事件

**事件I：申请学校**
> 一所名为"加州国际商业学院"的学校吸引了你的注意。他们的官网做得有模有样，声称"专为国际学生设计灵活课程"。你注意到他们的校舍照片都是从别的学校盗用的，但评论区的"毕业生"都在感谢学校"给了他们机会"。

**费用：** ¥50,000-80,000（首期"学费"）

**等待期：** 5-8回合

**事件II：获得I-20**
> 一封国际快递送到你手上。里面是一份I-20表格，证明你被录取了。信封里还有一张纸条："每学期需按时缴纳学费$4000以维持注册状态，无需到校上课。"

**事件III：面签**
相比旅游签，学生签通过率更高，但仍需检定：
```
成功条件：
- 持有I-20 + 现金证明（已满足）
- 英语 ≥ 5（能回答基础问题）

失败条件：
- 英语 < 3（无法回答基本问题）
```

**成功后的场景切换：**
```yaml
给予:
  - 学生签证（F-1）
  - I-20表格（道具）
  - 机票

清空: 所有常驻道具栏
保留:
  - 全局书籍池
  - 角色属性值

场景3起始状态:
  - 合法入境身份（学生签）
  - 现金: $1000-3000（首期学费已付）
  - 初始道具: 学校宿舍（前3回合免费）
  - 特殊机制: 
      - 每15回合需缴纳$4000"学费"维持身份
      - 不缴纳则身份失效，转为非法滞留
      - 从未需要"上课"，学校不存在实际课程
```

**场景3持续压力：**
野鸡大学的"学生"身份是一个持续的经济负担。你必须不断打工攒钱交学费，否则就会失去合法身份。这是一个精心设计的选择：花钱买一个临时的"合法"外衣，还是干脆黑下来？

### 直飞vs走线的权衡

| 对比项 | 走线（1→2→3） | 直飞（1→3） |
|-------|--------------|------------|
| 资金要求 | 低（¥15,000起） | 高（¥50,000+） |
| 时间成本 | 长（场景2约5-15回合） | 短（直接开始场景3） |
| 身份状态 | 非法入境 | 合法入境（有期限） |
| 场景2体验 | 完整体验 | 跳过 |
| 场景3起始 | 美国边境，需穿越 | 美国机场，直接入境 |
| 心理压力 | 高（死亡风险） | 中（经济压力） |

## 场景2 → 场景3：穿越边境

### 通用机制

场景2的核心目标就是到达美国边境并穿越。玩家需要积累足够的物资和信息，然后选择一种穿越方式。

**穿越前置：**
- 必须到达边境线（通过移动事件）
- 身体健康度 > 20（否则无法承受穿越压力）

### 路径A：向导带路

#### 事件ID: `act2_transition_guide`

**触发条件：**
- 持有 `guide` 标签道具（如 `guide_juan` 或 `smuggler_amir`）
- 或 现金 ≥ $500

**事件描述：**
> 你终于来到了边境墙前。高耸的金属墙延伸向远方，墙顶缠绕着带刺铁丝网。远处传来巡逻车的声音。你的向导说，他知道一个"缺口"，但需要等到天黑。

**向导选择：**

| 向导 | 费用 | 成功率 | 额外效果 |
|-----|------|-------|---------|
| 胡安（本地向导） | $500-800 | 75% | 失败后可重试 |
| 阿米尔（走私客） | $1500-2500 | 90% | 封锁随机事件干扰 |
| 临时向导（现场找） | $300-500 | 50% | 有30%概率被骗 |

**穿越检定：**
```
基础成功率: 根据向导类型
修正:
  + 体魄 ≥ 8: +10%
  + 风险意识 ≥ 6: +5%
  + 夜间行动: +10%
  - 身体健康 < 40: -15%

失败结果:
  - 被边境巡逻队抓获
  - 触发 `act2_captured` 事件链
  - 可能遣返或拘留
```

**成功后的场景切换：**
```yaml
扣除: 向导费用
清空: 所有常驻道具栏（场景2道具）
保留:
  - 全局书籍池
  - 角色属性值

场景3起始状态:
  - 非法入境身份
  - 现金: 剩余美元
  - 初始道具: 无（两手空空进入美国）
  - 初始地点: 边境小镇/沙漠边缘
  - Debuff: 【风声鹤唳】（初始压力Debuff）
```

### 路径B：穿越沙漠缺口

#### 事件ID: `act2_transition_gap`

**机制说明：**
这是一个**赌命型**切换路径——边境墙有一处损坏缺口，但墙后是茫茫沙漠。没有向导、没有补给，完全依靠生存本能。

#### 阶段I：寻找缺口

**触发事件：** `act2_town_find_gap`（寻找边境墙漏洞）

**执行：**
- 消耗：行动点 2
- 心理：-3（奔波的疲惫和不确定性）

**发现概率：**
- 基础成功率：20%
- 每次执行独立计算概率
- 可重复尝试直到成功

**成功结果：**
> 你发现了边境墙的缺口！在东南方向约两英里处，有一段墙体因暴雨冲刷而倒塌，足以让一个人钻过去。墙的另一侧是茫茫沙漠——没有向导、没有补给、没有退路。要么穿越沙漠，要么死在里面。

**失败结果：**
> 你找了一整天，问遍了镇上的每个人，但只得到了真假难辨的传闻和嘲讽的眼神。边境墙依然高不可攀。

**解锁条件：**
- 发现缺口后，解锁 `act2_cross_desert`（穿越沙漠缺口）事件
- 未发现的玩家无法选择穿越沙漠路径

#### 阶段II：穿越沙漠

触发事件 `act2_cross_desert`：

> 你来到传闻中的位置。确实，边境墙在这里有一个缺口——几根钢筋扭曲着指向天空，混凝土碎块散落在沙地上。但墙的另一侧只有无尽的黄沙，热浪在远处扭曲成诡异的形状。
> 
> 这是一个疯狂的选择。没有向导、没有补给、没有退路。要么穿越沙漠，要么死在里面。

**检定：**
```
沙漠生存检定:
  基础成功率: 40%
  修正:
    + 生存能力 × 4%（核心属性）
    + 风险意识 × 2%（辨别方向）
    + 持有 `water_container`: +15%（有水储备）
    + 持有 `compass_military`: +10%（不会迷路）
    
  最大成功率: 75%（沙漠永远有不可预测性）
```

**成功结果：**
> 你在烈日下走了整整两天。嘴唇干裂、视线模糊，但你凭借着惊人的意志力找到了一条干涸的河床——沿着它走，最终会到达公路。第三天清晨，你被一辆路过的皮卡发现。司机是个退休教师，他看了你一眼，什么也没问，只是递给你一瓶水。
> 
> "上来吧，"他说，"我送你到汽车站。"
> 
> 几个小时后，你登上了一辆开往加州的大巴。

**失败结果：**
- 【游戏结束】脱水倒在沙漠中，尸体永远不会被发现
- 【游戏结束】迷失方向，在沙漠中徘徊直到耗尽最后一丝体力

**场景3起始状态:**
```yaml
清空: 所有常驻道具栏
扣除: 
  - 现金 -20%（购买食物和水）
  - 身体健康 -15（沙漠消耗）
保留:
  - 全局书籍池
  - 角色属性值

场景3起始状态:
  - 地点: 加州（民主党州）
  - 身份: 非法入境
  - 现金: 剩余美元（扣除20%后）
  - 初始道具: 无
  - 非法入境身份，无特殊道具
```

**重要：不请向导的致命风险**
向导的存在不只是带路，更重要的是提供应急方案和应对意外的经验。独自穿越漏洞一旦失败，没有人救你。

**成功后的场景切换：**
```yaml
清空: 所有常驻道具栏
保留:
  - 全局书籍池
  - 角色属性值

场景3起始状态:
  - 非法入境身份
  - 现金: 剩余美元
  - 初始道具: 无
  - 非法入境身份，无特殊道具
```

### 路径C：直接攀爬

#### 事件ID: `act2_transition_climb`

**机制说明：**
墨西哥偷渡者的传统方式，对体魄要求极高，但完全免费。

**触发条件：**
- 体魄 ≥ 12
- 无向导道具（或有但选择不用）

**事件描述：**
> 你看着眼前的边境墙。约莫6米高，由一根根钢柱构成，烈日下泛着刺眼的光。钢柱表面光滑，温度高得能煎鸡蛋。没有铁丝网，只是光秃秃的钢柱和顶端的混凝土横梁。这是疯狂的做法，但你已经走到这一步了。

**攀爬检定：**
```
基础成功率: 40%
修正:
  + 体魄 12-14: +20%
  + 体魄 15-17: +35%
  + 体魄 18-20: +50%
  + 持有 `rope` 或 `hook`: +15%
  - 身体健康 < 50: -20%
  - 负重过多: -10%

最终成功率范围: 40% - 90%
```

**失败后果（致命）：**
| 失败类型 | 概率 | 后果 |
|---------|------|------|
| 摔落死亡 | 100% | 【游戏结束】运气不好，翻越时从墙顶摔落，当场死亡 |

**成功后果：**
- 成功翻越边境墙
- 健康 -20（钢柱烫手灼伤 + 跳下磕碰伤）
- 进入场景3

**重要：攀爬是玩命的赌注**
没有向导在下面接应、没有备用方案，一旦失手就是死亡。这是 poorest man's gamble（穷人的赌博）——因为你付不起向导费，所以拿命来赌。

**成功后的场景切换：**
```yaml
清空: 所有常驻道具栏
扣除: 身体健康 -20（假设）
保留:
  - 全局书籍池
  - 角色属性值

场景3起始状态:
  - 非法入境身份
  - 现金: 剩余美元
  - 初始道具: 无
  - 非法入境身份，无特殊道具
```

### 路径D：随机事件穿越

#### 事件ID: `rand2_border_miracle`

**机制说明：**
低概率触发的"奇迹"事件，可能帮助玩家跳过穿越难关。

**触发条件：**
- 在边境地区（已到达边境线）
- 随机触发（权重 5，约5%概率）
- 场景2回合数 ≥ 3（防止开局就触发）

**事件描述：**
> 一个穿制服的男人站在路边，手里拿着一叠纸。他看起来像是机场地勤，但这里离机场有几百公里。他看到你就招手："要机票吗？免费的。今晚飞洛杉矶。"

**选项分析：**

| 选项 | 条件 | 结果 |
|-----|------|------|
| **接受** | 无 | 50% 真的飞到场景3，50% 【游戏结束】 |
| **询问详情** | 智力 ≥ 6 | 检定是否识破骗局（识破后仍可选择） |
| **拒绝离开** | 风险意识 ≥ 5 | 安全但错过机会 |

**成功结果（50%概率）：**
```yaml
场景切换:
  清空: 所有常驻道具栏
  保留:
    - 全局书籍池
    - 角色属性值
  
  场景3起始状态:
    - 身份: ？？？（可能是假身份入境）
    - 现金: 剩余美元
    - 初始道具: 无
    - 非法入境身份，无特殊道具
```

**失败结果（50%概率）：【游戏结束】**
- 制服男子是贩毒集团的诱饵
- 你被带到沙漠深处
- 【游戏结束】你的尸体永远也不会被找到

**为什么这是赌博：**
因为你不愿意付向导钱，所以你把命交给一个随机出现的陌生人。这就是贪婪的代价。

## 设计原则总结

1. **不可逆选择**：每个切换决定都改变后续体验，没有"完美"路径
2. **风险与回报**：低成本路径高风险，高成本路径低风险
3. **身份差异**：合法入境vs非法入境在场景3有根本性体验差异
4. **资源重置**：强制清空制造紧张感，但保留属性让玩家有成长感
5. **叙事锚点**：每个切换都是故事高潮，值得精心文案
