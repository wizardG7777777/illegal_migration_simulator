# 核心设计原则（Design Principles）

> 本文档记录《去美国》游戏的核心设计哲学与硬性约束，是所有系统设计、事件设计、叙事设计的根本依据。

---

## 1. 角色设定

### 1.1 单主角制

- **整个游戏只有一个主角**：玩家创建的虚拟角色
- **一命通关**：角色死亡即游戏结束，不可复活、不可读档回到中途
- **身份延续**：玩家与角色绑定，体验完整的"一生"

### 1.2 角色与玩家的关系

- 第二人称叙事（"你"）
- 玩家通过选择塑造角色命运
- 没有"完美"选择，只有"代价"选择

---

## 2. 三幕结构核心差异

### 2.1 场景1：国内准备（新手教学）

**定位**：新手教学 + 决策准备期

**核心特征**：
| 维度 | 设计 |
|-----|------|
| **安全性** | 很难死亡，无随机致命事件 |
| **时间上限** | 50回合（软限制） |
| **预计时长** | 25回合 |
| **核心玩法** | 打工攒钱、属性提升、准备离境 |
| **退出方式** | 主动选择前往场景2/3，或被动触发"放弃"结局 |

**死亡机制**：
- 场景1本身**不会导致死亡**
- 若因极端情况死亡（如 debug 或特殊事件），**复用场景3的死亡结局文案**

**"放弃"结局**（特殊机制）：
- 触发条件：场景1停留回合数 ≥ 30
- 触发方式：弹出特殊事件，玩家选择"放弃前往美国的幻想"
- 结局性质：特殊结局（SPECIAL），视为通关一次
- 叙事意义：承认"留在国内也是一种选择"

**黑色幽默点**：
> "明明好好呆在国内一点事也没有，但人就是要去冒险。"

---

### 2.2 场景2：跨境穿越（生存考验）

**定位**：高风险生存挑战，死亡概率最高的场景

**核心特征**：
| 维度 | 设计 |
|-----|------|
| **安全性** | 极高风险，随机致命事件频发 |
| **移动方向** | 只能前进，**无退路** |
| **核心玩法** | 资源管理、路线选择、向导依赖 |
| **退出方式** | 成功进入场景3，或死亡 |

**压力来源**：
- **随机环境致命事件**：
  - 脚滑摔下山崖
  - 被鳄鱼/毒蛇袭击
  - 迷路后迷失在山林中脱水/饥饿而死
  - 暴雨、极端高温等恶劣天气
- **人为危险**：
  - 走私客背叛
  - 同伴冲突
  - 当地黑帮抢劫

**生存策略**：
- **跟随向导**：大幅降低风险，但需要支付费用
- **独自穿越**：高风险高回报，需高生存能力和运气

**场景2→3过渡死亡**（特定描述）：
| 穿越方式 | 死亡描述 |
|---------|---------|
| **漏洞穿越失败** | 倒在了茫茫沙漠中，尸体永远不会被发现 |
| **攀爬边境墙失败** | 从墙顶摔落，当场死亡 |
| **随机奇迹事件失败** | 被贩毒集团杀害，尸体被埋在沙漠深处 |

---

### 2.3 场景3：美国生存（结构性压迫）

**定位**：长期结构性压力，绿卡无望但不得不硬撑

**核心特征**：
| 维度 | 设计 |
|-----|------|
| **安全性** | 中等风险，但慢性死亡为主 |
| **时间长度** | 长期（60+回合） |
| **核心玩法** | 现金流管理、身份获取、多维度生存 |
| **胜利条件** | 获得绿卡（多种途径） |
| **失败方式** | 死亡（多种原因）、遣返、彻底黑户 |

**压力模型**：结构性恶性循环

```
需要钱生存 → 打工赚钱 → 身体/心理消耗 → 健康下降 → 
需要更多钱治病/恢复 → 更多打工 → 身体更差 → 死亡
```

**具体压力来源**：
1. **现金流压力**：
   - 生活成本（食品+住宿+出行）固定扣除
   - 通胀事件导致成本上涨
   - 收入不稳定（账号被封、被抢劫、突击检查）

2. **身份压力**：
   - 签证逾期倒计时
   - ICE突击检查风险
   - 假证件被识破风险

3. **身体消耗**：
   - 强化剂使用过度（副作用致死）
   - 长期过劳
   - 小病拖成大病（无医保）

4. **心理压力**：
   - 孤独、恐惧、绝望
   - 身份焦虑
   - 无法与家人联系

**黑色幽默点**：
> "赚到的钱和之前被宣传的完全不一样，但是生活需要的资金越来越多。"
> "绿卡看似有多种获取方式，但每条路都遥不可及。"

**拿绿卡的途径**（通关方式）：
- 庇护申请获批
- 与美国公民结婚
- 杰出人才（EB-1A）
- 投资移民（EB-5）
- 工作签证转正（H1B→绿卡）
- 多样性签证抽签（DV Lottery）

## 3. 叙事风格原则

### 3.1 黑色幽默基调

**核心矛盾**：
- **国内（场景1）**：安稳、平淡、但"没有梦想"
- **美国（场景2/3）**：危险、压迫、但"自由"

**叙事策略**：
- 不直接说教，让事件本身说话
- 用冷静克制的语气描述荒诞现实
- 让玩家自己体会到"这就是非法移民的真实"

**文本风格**：
- 第二人称（"你"）
- 短句、留白、不解释
- 避免煽情，保持疏离感

### 3.2 数值包装化（严禁直接显示数字）

| ❌ 禁止 | ✅ 正确 |
|-------|-------|
| "健康度-30" | "你感到一阵眩晕" |
| "现金+$120" | "口袋稍微鼓了一点" |
| "倒计时3回合" | "你感觉撑不了多久了" |
| "成功概率60%" | "看起来有希望，但也有风险" |

## 4. 硬性设计约束

### 4.1 入狱即终局

- **约束**：主角一旦入狱，失去再次出狱的可能
- **禁止**：保释、大赦、越狱、减刑
- **结果**：狱中终老（`death_prison`）或长期拘留（`special_detention`）

### 4.2 场景3二元结局

- **约束**：进入场景3后，结局只有两种可能：
  1. **留在美国**：通关结局（绿卡）或彻底黑户
  2. **回中国**：被遣返、自愿离境、主动回国
- **禁止**：转往第三国、加拿大、欧洲、失踪等

### 4.3 场景2单向流动

- **约束**：场景2只能前进，**无退路**
- **禁止**：返回场景1、原地停留过久

### 4.4 单结局原则

- **约束**：一个角色只有一个结局
- **禁止**：多结局共存、结局后复活

## 6. 设计意图速查

### 为什么要这样设计？

**Q: 为什么场景1要设置"放弃"结局？**  
A: 承认"留在国内也是一种选择"，强化"去美国"是一个主动的选择而非必然，增加反思空间。

**Q: 为什么场景2不能返回？**  
A: 模拟真实走线的不可逆性，一旦出发就没有回头路，制造紧张感和决策重量。

**Q: 为什么场景3是"结构性压迫"而非随机死亡？**  
A: 反映真实非法移民的生存状态：不是突发意外，而是慢性消耗，每天都是挣扎。

**Q: 为什么要禁止转往第三国的结局？**  
A: 保持叙事聚焦，避免分散主题。游戏讨论的是"去美国"，不是"离开中国"。
