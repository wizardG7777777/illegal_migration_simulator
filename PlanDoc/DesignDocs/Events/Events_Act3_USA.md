# 场景3事件：美国境内生存阶段

## 场景特征

**关键词**：现金流断裂、盘查恐惧、关系网搭建、长期压力与崩溃、身份洗白

**环境压力（Debuff）**：
- **【搜捕升级】**：每回合风声/热度自动上涨，隐蔽度自动下降
- **【大通胀时代】**：每回合现金消耗递增（通胀导致的物资涨价）
- 后果：拖得越久，移民局查得越严，黑户生存空间被压缩到极限，最终必然被捕
