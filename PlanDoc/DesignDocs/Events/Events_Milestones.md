# 里程碑/转折点事件设计

> 本文档定义游戏中的关键转折点和里程碑事件链。
> 基于 EventSystemArchitecture.md 中的事件链（CHAIN）类型设计。

---

## 1. 概述

里程碑事件是改变游戏走向的关键节点：
- **放弃线**：场景1特有的"放弃前往美国"结局路径
- **同伴变故**：场景2中同伴的死亡或背叛
- **身份获取**：场景3获得合法身份的主要路径

---

## 2. 场景1放弃结局线（Give Up Chain）

### 2.1 事件链总览

```
怀疑种子 → 现实压力 → 转机出现 → 最终决定
    ↓           ↓           ↓           ↓
 give_up_1   give_up_2   give_up_3   give_up_4
```

**触发条件**：场景1停留≥30回合后自动触发首事件

---

### 2.2 give_up_1_doubt（怀疑的种子）

| 字段 | 内容 |
|-----|------|
| **ID** | `give_up_1_doubt` |
| **链位置** | 节点1/4 |
| **类型** | RANDOM（自动触发） |

**触发条件**：
- 场景：act1
- 回合：≥30
- 心理健康度：< 60

**你看到了什么**：
> 深夜刷着朋友圈——国内的朋友已经升职加薪、买房结婚。而你还在打零工，为了那个遥远的梦想省吃俭用。"真的值得吗？"这个念头第一次如此清晰。

**可选行动**：

| 选项 | 条件 | 结果效果 |
|-----|------|---------|
| 继续坚持下去 | 无 | 心理+5（自我肯定），事件链继续 |
| 认真考虑放弃 | 智力≥8 | 解锁节点2，获得Debuff【动摇】（心理-3/回合） |
| 找朋友聊聊 | 社交≥6 | 随机结果：支持则心理+10继续，反对则加速放弃线 |

---

### 2.3 give_up_2_pressure（现实的压力）

## 3. 场景2同伴变故事件

### 3.1 companion_juan_captured（胡安被抓）

| 字段 | 内容 |
|-----|------|
| **ID** | `companion_juan_captured` |
| **类型** | RANDOM |

**触发条件**：
- 场景：act2
- 胡安为同伴且状态healthy
- 经过检查站/边境区域

**你看到了什么**：
> 探照灯亮起时，胡安推了你一把："跑！去前面的树丛！"你回头看见他被几个穿制服的人按在地上。

**可选行动**：

| 选项 | 条件 | 结果效果 |
|-----|------|---------|
| 回去救他 | 体魄≥12 | 50%几率两人一起被抓；50%几率逃脱但健康-20 |
| 听从他的话逃跑 | 无 | 胡安状态变为captured，你安全逃脱 |
| 独自逃跑 | 风险意识≥10 | 胡安状态变为captured，获得Debuff【愧疚】（心理-5/回合） |

**后续**：
- 胡安被抓后，5-10回合后可能触发 `companion_juan_returns`（支付赎金后返回）或永久失去

## 4. 场景3身份获取里程碑

### 4.1 庇护路径（Asylum Chain）

#### 事件链流程

```
咨询律师 → 准备材料 → 提交申请 → 等待面试 → 面试 → 等待结果 → 获批/被拒
    ↓           ↓           ↓           ↓         ↓          ↓
 asylum_1   asylum_2   asylum_3   asylum_4  asylum_5  asylum_6  [结局分支]
```

---

#### asylum_1_consult（咨询律师）

| 字段 | 内容 |
|-----|------|
| **ID** | `asylum_1_consult` |
| **链位置** | 节点1/6 |
| **类型** | FIXED |

**执行消耗**：行动点2，金钱$100-500（取决于律师质量）

**你看到了什么**：
> 唐人街地下室的"律师事务所"。华人律师打量着你："庇护？你先说说你的故事有多惨。"

**律师质量选项**：

| 选项 | 费用 | 效果 |
|-----|------|-----|
| 廉价律师 | $100 | 基础成功率+10%，材料质量-1 |
| 普通律师 | $300 | 基础成功率+25%，正常材料 |
| 名牌律师 | $500 | 基础成功率+40%，材料质量+1 |
| 自己研究 | $0（行动点+1） | 智力≥12时可达到普通律师效果 |

---

#### asylum_2_prepare（准备材料）

| 字段 | 内容 |
|-----|------|
| **ID** | `asylum_2_prepare` |
| **链位置** | 节点2/6 |

**执行消耗**：行动点3，耗时2-4回合

**材料准备检定**：
```
材料质量 = 基础值(5) + 律师加成 + 智力/3 + 随机(-2~+2)
材料质量≥8：面试时获得优势
材料质量<5：面试时可能暴露矛盾
```

**可选行动**：
- 仔细准备（智力≥10）：材料质量+2，耗时+1回合
- 快速准备：耗时-1回合，材料质量-1
- 伪造证据（风险意识<8）：50%几率获得优质材料，50%被发现造假直接拒签

---

#### asylum_3_submit（提交申请）

| 字段 | 内容 |
|-----|------|
| **ID** | `asylum_3_submit` |
| **链位置** | 节点3/6 |

**执行消耗**：行动点1

**你看到了什么**：
> 移民局大楼前排着长队。你把厚厚的材料递进窗口，工作人员面无表情地盖了个章。"等通知。"

**结果**：
- 获得收据（临时合法身份，可合法打工90天）
- 进入等待期（10-30回合后触发面试）

---

#### asylum_4_wait_interview（等待面试）

| 字段 | 内容 |
|-----|------|
| **ID** | `asylum_4_wait_interview` |
| **链位置** | 节点4/6 |
| **类型** | CHAIN（自动推进） |

**等待期间**：
- 可正常工作
- 心理压力+3/回合（不确定性）
- 随机触发事件：`asylum_check_in`（定期报到）

---

#### asylum_5_interview（庇护面试）

| 字段 | 内容 |
|-----|------|
| **ID** | `asylum_5_interview` |
| **链位置** | 节点5/6 |
| **类型** | FIXED（关键检定） |

**你看到了什么**：
> 面试官面无表情地翻着你的材料。"你说你在国内因为参加抗议被拘留——"他抬头，"具体是哪一天？"

**面试检定**：
```
面试得分 = 材料质量 + 智力/2 + 英语/3 + 随机(-5~+5)
得分≥20：通过
得分15-19：补充材料
得分<15：拒绝
```

**关键选项**：
- 诚实回答：基于材料质量得分
- 补充细节（智力≥12）：+2分
- 紧张忘词（心理状态<40）：-3分

---

#### asylum_6_result（庇护结果）

| 字段 | 内容 |
|-----|------|
| **ID** | `asylum_6_result` |
| **链位置** | 节点6/6（终点） |

**面试后5-10回合触发**

**通过**：
- **结局：success_asylum**
- 获得工卡，1年后可申请绿卡

**拒绝**：
- 进入遣返程序（可选择上诉，成功率20%）
- 或触发 `special_underground`（转入地下）

## 5. 其他身份获取路径简述

### 5.1 技术移民路径
- 找到雇主担保
- H1B抽签（纯运气，30%成功率）
- 获批后排队等绿卡（3-5年）

### 5.2 EB-5投资移民
- 投资$80万-100万
- 创造10个就业岗位
- 直接获得绿卡

### 5.3 抽签移民（DV Lottery）
- 纯随机事件触发
- 每年10月可参与
- 中签率约0.5%

### 5.4 特殊人才（EB-1A）
- 属性智力≥18
- 完成特定成就事件链
- 难度最高但速度最快
