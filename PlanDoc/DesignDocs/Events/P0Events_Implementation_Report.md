## 事件详细说明

### 1. act3_living_cost - 每月生活成本自动扣除 ⭐核心机制

**位置**: `Events_Act3_USA.md` 第1887行起

**关键数值配置**:
```yaml
触发时机: 每30回合（每月1日）
食品成本: $400（固定）
住宿成本:
  - 无住所: $0（但获得【露宿街头】Debuff）
  - 收容所: $0（心理-3/回合）
  - 合租: $600
  - 公寓: $1,200
  - House: $2,500
出行成本:
  - 无交通工具: $0（工作限制）
  - 公交卡: $100
  - 电动车: $50
  - 二手车: $350
  - 特斯拉: $600
```

**实现要点**:
- 系统强制触发，不可跳过
- 与道具系统（`lodging`、`transport`标签）联动
- 支持通胀Debuff影响
- 现金不足触发匮乏状态

### 2. act3_inflation_trigger - 通胀Debuff触发事件

**位置**: `Events_Act3_USA.md` 第2000行起

**层级效果**:
| 层级 | 名称 | 生活成本 | 打工收入 | 心理消耗 |
|------|------|---------|---------|---------|
| 1 | 【物价上涨】 | +10% | 无 | 无 |
| 2 | 【通货膨胀】 | +20% | -10% | 无 |
| 3 | 【恶性通胀】 | +30% | -15% | -2/回合 |

**触发条件**:
- 场景3回合 > 20
- 每10回合20%概率
- 最多触发3次

**实现要点**:
- 三层可叠加Debuff系统
- 新闻公告形式叙事
- 与生活成本扣除系统联动

---

### 3. rand3_ice_raid - ICE突袭完整流程

**位置**: `Events_Act3_USA.md` 第2112行起

**触发概率计算**:
```
基础概率 = 5%
+ (移民执法压力强度 × 2%)
+ (露宿街头 ? 5% : 0%)
+ (【逾期滞留】标记 ? 5% : 0%)
```

**三阶段设计**:
1. **阶段1** - 突袭发生：凌晨砸门，"ICE! OPEN UP!"
2. **阶段2** - 玩家选择：
   - 躲藏（生存能力≥10）
   - 假装住户（社交≥12 + `fake_lease`）
   - 逃跑（风险意识≥8）
3. **阶段3** - 被捕后：触发遣返流程链

**实现要点**:
- 三种选择检定平衡
- 与压力Debuff联动
- 被捕后进入detained状态

### ~~4. act3_deportation_process - 遣返流程链~~ 【已废弃/未实现】

> **废弃原因**：根据设计原则更新，场景3采用"入狱即终局"约束（无保释、无遣返程序）。被捕后直接触发`ending_arrested`入狱终局结局。

**原设计内容（仅供参考）**：

**位置**: `Events_Act3_USA.md` 第2296行起

**链节点**:
1. **拘留中心**（3-5回合）
   - 每回合心理-10
   - 可选：联系律师（-$1000）/等待/认罪
   
2. **移民法庭**
   - 有律师：50%暂缓 / 50%遣返
   - 无律师：10%暂缓 / 90%遣返
   
3. **结局分支**
   - 遣返：触发结局【遣返回国】
   - 暂缓：获得临时保护身份，继续游戏
   - 庇护申请中：进入庇护法庭流程

### 5. act3_visa_overstay - 学签/旅游签过期事件

**位置**: `Events_Act3_USA.md` 第2471行起

**旅游签**:
- 6回合倒计时
- 第2回合警告，第1回合紧急警告
- 到期自动转为undocumented
- 获得【逾期滞留】标记（ICE概率+5%）

**学签**:
- 每15回合需支付$4,000学费（共4期）
- 入学时尾款约$1,389（如分期）
- 未支付则身份失效
- 提前2回合警告

**实现要点**:
- 倒计时系统清晰
- 身份转换状态正确
- 与ICE突袭概率联动

---

### 6. act3_lodge_shelter - 收容所床位

**位置**: `Events_Act3_USA.md` 第2619行起

**基本效果**:
- 获得道具：`lodging_shelter`
- 成本：$0
- 健康恢复：+2/回合
- 心理消耗：-3/回合
- ICE突袭概率：+10%

**续期机制**:
- 每7回合需续期
- 社交检定≥5成功
- 失败则转为露宿街头

**实现要点**:
- 与露宿街头状态区分
- 定期申请机制合理
- 低成本但高压力平衡
