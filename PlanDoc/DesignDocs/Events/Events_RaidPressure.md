# 特朗普移民政策声明事件集

> 本文档定义场景3（美国境内）中，由唐纳德总统政策声明触发的移民执法压力事件。
> 
> **核心设计原则**：
> - **只在场景3生效**：压力数值仅在 Act3State 中存储
> - **政策声明触发**：通过"唐纳德总统宣布XXX"形式的事件增加压力
> - **单向增长**：压力只增不减，玩家无法左右
> - **叙事包装**：每个事件都有明确的政治新闻背景

---

## 1. 系统设计

### 1.1 压力存储位置

```typescript
// 只在场景3状态中存储
interface Act3State {
  // ... 其他字段
  
  // 移民执法压力（场景3特有）
  enforcementPressure: {
    current: number;             // 0-100
    history: {
      turn: number;
      policyEvent: string;       // 政策事件ID
      description: string;       // "唐纳德总统宣布XXX"
      amount: number;
    }[];
  };
}
```

### 1.2 压力增长方式

| 增长方式 | 说明 | 频率 |
|---------|------|------|
| **政策声明事件** | 随机触发的总统宣布 | 不定期 |
| **基础增长** | 每回合自动小幅增加 | 每回合 +1 |

### 1.3 压力等级

| 等级 | 范围 | 描述 | 影响 |
|-----|------|------|------|
| 绿色 | 0-25 | 相对平静 | 正常随机事件池 |
| 黄色 | 26-50 | 开始关注 | ICE检查频率增加 |
| 橙色 | 51-75 | 严查阶段 | 工作场所突袭概率大增 |
| 红色 | 76-100 | 紧急状态 | 大规模搜捕，随时可能游戏结束 |

## 3. 基础压力增长

除了政策声明事件，场景3还有基础压力增长：

```typescript
// 每回合开始时的基础增长（Act3）
function applyBasePressureGrowth(character: Character): void {
  if (character.status.currentScene !== 'act3') return;
  
  const act3State = character.sceneState.act3;
  const currentPressure = act3State.enforcementPressure.current;
  
  // 基础增长 +1
  let growth = 1;
  
  // 如果已触发过政策事件，增长加速
  if (act3State.enforcementPressure.history.length > 0) {
    growth += 1;
  }
  
  // 接近终局时增长更快
  if (currentPressure > 70) {
    growth += 1;
  }
  
  act3State.enforcementPressure.current = Math.min(100, currentPressure + growth);
}
```

## 4. 压力对游戏的影响

### 4.1 压力等级效果

| 压力 | ICE检查频率 | 打工风险 | 心理消耗 |
|-----|------------|---------|---------|
| 0-25 | 正常 | 低 | 正常 |
| 26-50 | +50% | 中 | 每回合-1 |
| 51-75 | +100% | 高 | 每回合-2 |
| 76-100 | +200% | 极高 | 每回合-3，可能直接触发逮捕 |

### 4.2 压力与结局关联

- **压力 < 50**：正常结局判定
- **压力 50-75**：结局被降级（好结局→普通结局）
- **压力 > 75**：大概率触发被捕/遣返结局
- **压力 = 100**：游戏强制结束（被捕）
