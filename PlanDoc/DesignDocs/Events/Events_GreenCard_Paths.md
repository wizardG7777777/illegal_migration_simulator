# 绿卡路径事件文档

> 本文档定义场景3（美国生存）中通往绿卡的三条主要路径的具体事件。

---

## 路径A：唐纳德总统金卡

### 前置事件

| 属性 | 值 |
|------|-----|
| **事件ID** | `act3_hear_gold_card` |
| **事件类型** | RANDOM |
| **触发条件** | 场景3任意回合，现金 >= $500,000 |

**事件描述**：

> 你在华人论坛上看到一个帖子：
> "特朗普总统即将推出金卡计划，500万美元直接拿绿卡，无需排队，无需审查。"
> 你盯着那个数字，心跳加速。

**系统效果**：
- 解锁事件 `act3_buy_gold_card`

---

### 主事件：购买金卡

| 属性 | 值 |
|------|-----|
| **事件ID** | `act3_buy_gold_card` |
| **事件类型** | FIXED |
| **触发条件** | 现金 >= $1,000,000，已触发前置事件 `act3_hear_gold_card` |
| **物品槽位** | 无 |

**事件描述**：

> 你被带到一个高档写字楼。律师彬彬有礼地递来合同。
> "签下这份文件，您就是美国的合法永久居民了。"
> 没有面试，没有审查，只有刷卡机的声音。

**可选行动**：

| 选项 | 条件 | 结果 |
|------|------|------|
| **签下合同** | 现金 >= $1,000,000 | 触发结局【黄金门票】 |
| **再考虑考虑** | 无 | 事件关闭，可随时重新触发 |

**系统效果**：

| 效果 | 数值 |
|------|------|
| 现金 | -$1,000,000 |
| 结局 | 触发通关结局【黄金门票】 |

## 路径B：政治庇护（事件链）

### 启动事件

| 属性 | 值 |
|------|-----|
| **事件ID** | `chain_asylum_start` |
| **事件类型** | CHAIN（链式事件-启动） |
| **触发条件** | 回合 >= 20，现金 >= $200 |
| **物品槽位** | 无 |

**事件描述**：

> 你找到了一位专门做庇护申请的律师。
> "告诉我你的故事，"他说，"越惨越好。"
> 你编造了一个故事...

**可选行动**：

| 选项 | 条件 | 结果 |
|------|------|------|
| **启动庇护申请** | 现金 >= $200 | 支付启动费，5回合后触发阶段1 |
| **放弃** | 无 | 事件关闭 |

**系统效果**：

| 效果 | 数值 |
|------|------|
| 现金 | -$200 |
| 状态 | 启动政治庇护事件链 |
| 倒计时 | 5回合后自动触发 `chain_asylum_1` |

---

### 阶段1：材料准备

| 属性 | 值 |
|------|-----|
| **事件ID** | `chain_asylum_1` |
| **事件类型** | CHAIN |
| **触发条件** | 启动事件已执行，间隔5回合 |
| **物品槽位** | 无 |

**事件描述**：

> 律师点点头："不错，但还需要更多细节。我们需要'证据'。"
> 你需要准备假报警记录、医院证明...

**判定逻辑**：

| 条件 | 成功率修正 |
|------|-----------|
| 基础成功率 | 50% |
| 智力 >= 10 | +20% |
| 风险意识 >= 8 | +10% |
| **最高成功率** | **80%** |

**可选行动**：

| 选项 | 条件 | 结果 |
|------|------|------|
| **继续准备材料** | 现金 >= $300 | 进行判定 |
| **放弃申请** | 无 | 事件链终止 |

**判定结果**：

| 结果 | 描述 | 后续 |
|------|------|------|
| **成功** | 材料通过审核 | 5回合后触发 `chain_asylum_2` |
| **失败** | 材料被认为不够充分 | 需重新执行阶段1（重新付费$200） |

**系统效果**：

| 效果 | 数值 |
|------|------|
| 现金 | -$300（材料费） |
| 社交 | -2（永久损失） |

## 设计备注

### 三条路径对比

| 路径 | 资金需求 | 时间需求 | 风险 | 属性要求 | 道德色彩 |
|------|----------|----------|------|----------|----------|
| **金卡** | 极高($1M) | 即时 | 无 | 无 | 中性 |
| **庇护** | 中等($2K+) | 短(15回合) | 高 | 智力/风险意识 | 灰色 |
| **U签证** | 低($1K+) | 即时 | 低 | 社交 | 黑色幽默 |

### 平衡性说明

1. **金卡**：纯粹的资金门槛，适合高风险高回报的赚钱路线玩家
2. **庇护**：属性检定机制，适合高智力角色，但有失败遣返风险
3. **U签证**：无失败概率，缴纳费用即成功，但需要先"成为受害者"

### 关联事件

- 庇护路径的遣返结果会触发 `Events_Terminal.md` 中的遣返结局
- 金卡结局和U签证结局均为通关结局，会触发 `Events_Milestones.md` 中的结算流程
