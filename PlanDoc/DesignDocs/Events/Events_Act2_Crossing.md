# 场景2事件：跨境穿越阶段

## 场景特征

**关键词**：极端环境、迷路风险、同伴背叛、武装抢劫、生存极限

**环境压力（Debuff）**：
- **【环境恶化】**：每回合身体健康度大幅扣除（饥饿/疾病），物资快速消耗
- 后果：拖得越久，身体越垮，遇到突发事件（黑帮/天气）时的死亡率直线上升
