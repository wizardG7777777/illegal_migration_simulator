## 文件 1: NarrativeDocs/README.md

```markdown
# NarrativeDocs - 叙事文档

本文档目录包含《去美国》文字冒险游戏的所有叙事相关内容。

## 目录结构

```
NarrativeDocs/
├── StoryOutline/          # 整体剧情大纲
├── Dialogues/             # 对话文本库
├── Lore/                  # 碎片化信息内容（待建设）
└── CharacterStories/      # 角色背景故事（待建设）
```

## 使用指南

- **StoryOutline/**: 定义三幕结构的叙事主线
- **Dialogues/**: 包含所有游戏中展示给玩家的文本
- **Lore/**: 支撑世界观但不直接展示的背景信息
- **CharacterStories/**: 角色背景和NPC档案

## 文案风格规范

1. **基调**: 冷静克制的旁观者视角，带黑色幽默
2. **视角**: 第二人称（"你"）
3. **避免**: 直接呈现数值（用"感觉好多了"代替"心理健康度+10"）

## 与 DesignDocs/ 的关系

| NarrativeDocs | 对应 DesignDocs | 关系 |
|--------------|----------------|------|
| StoryOutline | DesignPrinciples | 大纲 ↔ 约束 |
| Dialogues | Events_Act*.md | 叙事文本 ↔ 机制表格 |
| CharacterStories | CharacterInfo | 背景故事 ↔ 属性系统 |
| Lore | World/ | 碎片化信息 ↔ 场景设定 |
```

# Dialogues - 对话文本库

本文档目录包含游戏中所有展示给玩家的文本内容。

## 文档结构

| 文件 | 内容 | 状态 |
|-----|------|------|
| Templates/ | 对话模板和写作规范 | 待创建 |
| Act1_Dialogues.md | 场景1事件文案 | 待创建 |
| Act2_Dialogues.md | 场景2事件文案 | 待创建 |
| Act3_Dialogues.md | 场景3事件文案 | 待创建 |
| EndingTexts.md | 所有结局文案 | 待创建 |
| CommonTexts.md | 通用文本 | 待创建 |

## 写作规范

### 1. 事件描述格式

```markdown
### 事件名称 (ID: `event_id`)

**你看到了什么**（场景描写，50-100字）

**可选行动**：
1. **选项A**（条件：属性≥X）→ 结果描述
2. **选项B** → 结果描述
3. **选项C**（需道具：xxx）→ 结果描述
```

### 2. 数值包装原则

| ❌ 不要 | ✅ 要 |
|--------|------|
| "心理健康度+10" | "你感到内心平静了一些" |
| "现金-500" | "你数出五张钞票递过去" |
| "体魄检定失败" | "你感到力不从心" |

### 3. 文案长度规范

- **事件描述**: 50-100字
- **选项文本**: 10-20字
- **结果描述**: 30-50字
- **结局文本**: 100-200字

## 与事件机制的对应关系

Dialogues/ 中的文案对应 DesignDocs/Events/ 中的事件机制：

- 事件ID必须与 Events_Act*.md 中的ID对应
- 选项条件必须与事件定义一致
- 数值变化结果必须与机制表格一致
```

### 示例 1: 打工事件

```markdown
### 连锁超市夜班 (ID: `act1_work_supermarket`)

**深夜的仓库，荧光灯发出轻微的电流声。**

货架间的通道像迷宫一样延伸。你推着叉车穿过一排排商品，
将到货的货物摆上货架。凌晨三点的超市静谧得只剩下你的
脚步声和设备运转的嗡嗡声。

---

**可选行动:**

#### 选项 1: 认真工作

**条件**: 无

**你决定:**
你专注于手头的工作，尽量不出差错。尽管身体疲惫，
但想着能多攒一点钱，你就咬牙坚持。

**结果:**
天亮交班时，主管难得地对你点了点头。你接过工钱，
感觉虽然身体酸痛，但至少钱包鼓了一些。

---

#### 选项 2: 摸鱼休息

**条件**: 风险意识≥8

**你决定:**
你找了个监控死角，靠着货架休息了几分钟。
你警惕地观察着周围的动静，随时准备回到岗位。

**结果:**
你成功偷了个懒，体力恢复了一些。虽然少做了些活，
但至少没被主管发现。

---

#### 选项 3: 向老员工打听

**条件**: 社交≥10

**你决定:**
趁着休息的空档，你凑到老员工身边，套近乎地递了根烟，
询问关于"出去"的事情。

**结果:**
老员工压低声音说了些什么，你记下了一些有用的信息。
